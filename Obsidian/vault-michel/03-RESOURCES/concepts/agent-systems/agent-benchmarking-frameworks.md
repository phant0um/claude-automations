# Agent Benchmarking Frameworks

## Definição

"Agentic" não é binário — é um espectro. Benchmarks precisam medir grau de autonomia, não boolean. IC3 paper mostra que maioria das plataformas de AI agents tem 0 evidência de trading on-chain autônomo. Benchmarks emergentes medem capacidade de tool use, multi-step reasoning, e self-correction.
