# Agent Oversight Layers

## Definição

Oversight como camada de arquitetura, não add-on. Agents autônomos precisam de controle de information-flow (IFC), layers de oversight (OpenSigil), e confidentiality garantida (MosaicLeaks). A tese: autonomia e oversight são eixos ortogonais — mais autonomia exige mais oversight, não menos. Vercel Passport/Connect produtiza governance como default de platform.
