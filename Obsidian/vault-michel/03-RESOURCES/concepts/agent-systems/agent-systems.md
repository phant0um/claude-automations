# Agent Systems

Top-level redirect: the field of designing and running multi-component AI systems where LLMs act as autonomous actors.

→ See [[03-RESOURCES/concepts/agent-systems/_index]] for the full index of patterns, components, and vault-specific implementations.

## One-liner

Agent systems = architectures where one or more AI agents perceive, reason, act, and collaborate to complete goals that exceed what a single model call can handle.
