# Automated Research & Autoresearch

Agents that autonomously gather, synthesize, and validate research.

## Autoresearch Workflow

1. **Query Definition** — What to research?
2. **Source Discovery** — Where to look?
3. **Data Collection** — Scrape, parse, standardize
4. **Synthesis** — Combine findings, spot contradictions
5. **Validation** — Cross-check sources, rate confidence
6. **Presentation** — Formatted report

## Agent Types

### Searcher
- Query construction and execution
- Source ranking
- Link extraction

### Scraper  
- Content extraction from pages
- HTML/PDF parsing
- Data normalization

### Synthesizer
- Pattern extraction across sources
- Contradiction detection
- Summary generation

### Validator
- Cross-source verification
- Confidence scoring
- Citation tracking
