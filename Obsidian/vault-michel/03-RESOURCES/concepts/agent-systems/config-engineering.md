# Config Engineering

## Definição

Config e skills são a alavanca de performance mais subutilizada em agent systems. Instruções bem estruturadas > tamanho do modelo (SkillsBench: Haiku com bons workflows > Opus sem eles). Mas config ruim é pior que nenhuma — CLAUDE.md genérico e inflado é o maior problema de produtividade. Menos é mais: < 100 linhas, específico, acionável.
