# Content Automation & Pipeline Orchestration

End-to-end content workflows from ideation to publishing.

## Pipeline Stages

1. **Research** — Autoresearch agent gathers data
2. **Outline** — Structure and key points
3. **Writing** — Content generation with voice
4. **SEO Optimization** — Keywords, meta, structure
5. **Editing** — Review and refinement
6. **Publishing** — Multi-platform deployment

## Tools

### Claude Cowork
- Collaborative editing
- SEO optimization built-in
- Multi-platform publishing
- Analytics integration

### Perplexity
- Real-time research integration
- Source attribution
- Fact verification

## Use Case: X/Twitter Pipeline
- Daily idea generation
- Content clustering
- Thread authoring
- Cross-posting
- Analytics
