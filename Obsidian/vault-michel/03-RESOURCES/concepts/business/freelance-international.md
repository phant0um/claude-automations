# Freelance International

Estratégia para freelancing em dólares no mercado internacional via plataformas como Upwork.

## Path B (mais rápido): Freelancing/turnkey

- Especialidade narrow vende mais que "AI in general"
- Upwork → reviews → raise rate → Toptal/Fiverr
- Free test on client's clip removes half the doubt

## Rates (2026)

| Level | Freelance ($/hr) | Turnkey |
|-------|------------------|---------|
| Junior | $50-80 | from ~$10k |
| Middle | $80-120 | — |
| Senior | $120-200+ | up to $250k+ |

## 90 days plan

Week 1: Colab setup → Weeks 2-3: demos → Week 4: GitHub + portfolio → Week 5: Upwork profile → Weeks 6-10: proposals → Weeks 11-13: deliver + reviews
