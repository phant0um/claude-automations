# MCP Server Curation

The practice of selecting, evaluating, and maintaining a personal stack of MCP servers for Claude. Not all MCPs are equal — value depends on existing tool usage, workflow type, and setup friction.

## Evaluation Axes

- **Setup friction:** Official Anthropic Connector (seconds) vs. Claude Code + API key (minutes)
- **Dependency:** Only useful if already on that platform (e.g., Zapier)
- **Composability:** Google Suite MCPs compound when all three are active simultaneously
- **Coverage:** Finance, productivity, creative, research categories
