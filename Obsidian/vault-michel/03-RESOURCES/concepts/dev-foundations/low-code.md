# Low-code / No-code

Plataformas que permitem desenvolvimento de software com mínimo (low-code) ou zero (no-code) código manual via interfaces visuais (drag-and-drop, fluxos, configuração).

## Diferença

| | Low-code | No-code |
|---|---|---|
| Código manual | possível, opcional | nenhum |
| Audience | devs + power users | citizen developers |
| Flexibilidade | alta | limitada |
| Curva | média | baixa |

## Exemplos

**Low-code:**
- [[03-RESOURCES/entities/Node-RED|Node-RED]] (IBM, fluxos)
- OutSystems, Mendix (apps empresariais)
- PowerApps, Zoho Creator

**No-code:**
- Webflow, Squarespace (sites)
- Airtable, Notion (DBs/docs)
- Bubble (web apps)
- Zapier, Make (automação)
- Glide (mobile via spreadsheet)

## Casos de uso

- Prototipagem rápida
- Automação de processos internos
- Dashboards e formulários
- Integrações entre SaaS
- MVPs

## Limitações

- Vendor lock-in
- Escalabilidade
- Customização avançada
- Performance em apps grandes
- Custo de licenças cresce com uso
