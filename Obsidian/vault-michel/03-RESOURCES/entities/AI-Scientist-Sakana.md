# AI Scientist (Sakana AI)

**GitHub:** github.com/SakanaAI/AI-Scientist · SakanaAI/AI-Scientist-v2

Primeiro sistema para descoberta científica totalmente automática, criado pela Sakana AI — pipeline completo de hipótese → experimentos → manuscrito → revisão por pares, tudo autônomo.

## Contribuições relevantes

- v1: primeiro sistema end-to-end de descoberta científica autônoma (template-based)
- v2: descoberta científica via agentic tree search, remove dependência de templates pré-definidos
- Parte do ecossistema "awesome-autoresearch" — um dos sistemas de referência na categoria Research Agent Systems
- Benchmark de referência para sistemas autônomos de pesquisa científica
