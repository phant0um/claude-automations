# AIDE (Weco AI)

**GitHub:** github.com/WecoAI/aideml · Weco AI

AIDE (AI Software Engineer) é o agente de ML engineering autônomo da Weco AI que usa tree-search para geração iterativa e avaliação de código, indexado no vault como referência de benchmark em autoresearch.

## Contribuições relevantes

- Abordagem: tree-search iterativo para geração de código + avaliação automática de qualidade
- Parte do ecossistema "awesome-autoresearch" — categoria de agentes de ML engineering
- Plataforma Weco Cloud para deployment do AIDE em produção
- Benchmark de referência para coding agents autônomos em tarefas de ML
