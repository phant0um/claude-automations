# AlexFinn

**Handle:** @AlexFinn (X/Twitter)

Criador de conteúdo com foco em uso prático de IA para produtividade. Popularizou a técnica do **Prompt Reverso** como principal habilidade de IA que um usuário pode aprender.

## Contribuições relevantes

- **Prompt Reverso** — técnica de fazer a IA perguntar ao usuário o que ela precisa saber para ajudá-lo; inverte o fluxo padrão de chat e multiplica a utilidade do agente.
- **Planning Interrogation** — variante do Prompt Reverso para features completas: forçar o agente a fazer 100+ perguntas antes de escrever qualquer código, gerando um plano que permite execução autônoma com criação automática de issues no Linear.
