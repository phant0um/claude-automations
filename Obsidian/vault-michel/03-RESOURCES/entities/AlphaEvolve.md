# AlphaEvolve

**Organização:** Google DeepMind

Sistema evolutivo do Google DeepMind que usa LLMs para descobrir novos algoritmos, combinando busca evolutiva com geração de código por modelos de linguagem.

## Contribuições relevantes

- Descobriu algoritmo de multiplicação de matrizes mais eficiente que os melhores algoritmos humanos conhecidos
- Blend de busca evolutiva + geração de código LLM: LLM propõe mutações, fitness function avalia, melhores sobrevivem
- Instância concreta do paradigma "autoresearch": LLM como gerador de hipóteses em loop autônomo
- Demonstra que LLMs podem fazer contribuições originais em matemática/algoritmos, não apenas aplicar conhecimento existente
