# Ashwin Gopinath

Handle: @ashwingop on X.

Founder/builder at [[03-RESOURCES/entities/Sentra-AI]] — building enterprise general intelligence: a shared intelligence/memory layer that sits on all communication channels, knowledge bases, and agent traces to understand how everyone in an organization actually works.

## Key Ideas

- **Company Brain** framework: three-layer memory model (factual, interaction, action) for organizations. See [[03-RESOURCES/concepts/pkm-obsidian/company-brain]].
- **Factual memory** as Layer 1: provenance, permissions, ownership, freshness, relationships between artifacts. See [[03-RESOURCES/concepts/llm-ml-foundations/factual-memory]].
- **Semantic file system** as the required storage model — relationships around an artifact matter as much as the artifact itself. See [[03-RESOURCES/concepts/pkm-obsidian/semantic-file-system]].
- Memory must emerge from individual workspaces outward, not be imposed top-down.
- "A knowledge base waits. Memory participates."
