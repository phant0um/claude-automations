# Boris Cherny

Creator of [[03-RESOURCES/entities/Claude Code]] at Anthropic. Published internal workflow patterns for Claude Code that became widely referenced in the developer community.

## Known for

- Designing the Claude Code CLI and its agentic workflow model
- Sharing internal team workflows (plan mode, subagent strategy, self-improvement loop, autonomous bug fixing) that informed the community's best practices for CLAUDE.md configuration

## The 13 Principles (named and numbered)

1. Plan Mode Default — 2. Subagent Strategy — 3. Self-Improvement Loop — 4. Verification Before Done — 5. Demand Elegance (Balanced) — 6. Autonomous Bug Fixing — 7. Plan First — 8. Verify Plan, Track Progress — 9. Explain Changes & Results — 10. Capture Lessons — 11. Simplicity First — 12. No Laziness — 13. Minimal Impact
