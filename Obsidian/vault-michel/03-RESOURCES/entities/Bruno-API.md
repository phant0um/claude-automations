# Bruno API

## Descrição

Cliente de API open-source, alternativa ao Postman. Integração com AI workflows (Cursor, Claude Code, agents). Suporta coleções, OAuth 1.0/2.0, git integration, e Bruno workflows que trazem agent skills para API testing. Útil para dev agents que precisam testar APIs como parte de loops de desenvolvimento.
