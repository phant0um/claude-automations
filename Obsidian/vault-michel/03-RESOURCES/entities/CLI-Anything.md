# CLI-Anything

## O que é

Framework do HKUDS que **gera CLIs dinamicamente** para qualquer software. Paradigma agent-native: ao invés de adaptar agente a UI/API existente, o framework cria uma CLI sob medida para que o agente consuma.

URL: `clianything.cc`

## Premissa

Agentes operam melhor com CLI textual do que com GUI. Mas a maioria do software não tem CLI completa. CLI-Anything resolve gerando a interface CLI on-demand, tornando qualquer app "agent-native".

## Por que importa

Inverte a relação humano-centric → agent-centric:
- Tradicional: software desenhado pra humano, agente adapta
- CLI-Anything: software ganha uma face CLI gerada, agente consome nativamente

Aponta para infraestrutura de "agent-native software" — categoria emergente.
