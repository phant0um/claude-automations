# Claude Design

**Organização:** Anthropic — equipe de design

Equipe de design da Anthropic responsável pelo sistema de design Claude, incluindo component library, design tokens e padrões de interação para interfaces de IA.

## Contribuições relevantes

- Mantém o sistema de design oficial para interfaces Claude: componentes, tokens, guidelines de UX
- Define padrões de interação para AI chat UIs (estados de loading, feedback de erro, progressive disclosure)
- Referência para designers que trabalham com integração de IA em produtos
- Citado no vault em fonte sobre uso de Claude Code para design de marca
