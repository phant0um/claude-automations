# Claude Haiku 4.5

Modelo mais rápido e barato da família Claude, otimizado para throughput alto e tarefas simples.

## O que é / What it is

Claude Haiku 4.5 (ID: `claude-haiku-4-5-20251001`) é projetado para pipelines de alto volume onde custo e velocidade dominam sobre profundidade de raciocínio. Ideal para classificações simples, extração de dados estruturados, triagem de documentos e subagentes paralelos que executam tarefas atômicas.

## Relevância para o vault

Michel usa Haiku como motor de subagentes de baixo custo no `batch-ingest` e em pipelines paralelos de processamento de Clippings, mantendo custo de API baixo sem sacrificar throughput.
