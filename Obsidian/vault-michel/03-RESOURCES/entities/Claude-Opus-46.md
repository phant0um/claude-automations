# Claude Opus 4.6

**Type:** Large language model by [[03-RESOURCES/entities/anthropic]]

## Role in AAR Research

Claude Opus 4.6 is the model that powers the **Automated Alignment Researcher (AAR)** agents in [[03-RESOURCES/sources/ml-research-papers/automated-weak-to-strong-researcher]]. Nine parallel Claude Opus 4.6 agents ran for 800 cumulative hours achieving PGR = 0.97 on the weak-to-strong supervision problem.

## Context

- Predecessor to [[03-RESOURCES/entities/Claude-Opus-47]] (launched April 16, 2026)
- Used for research agents in the Anthropic Fellows Program
- Also used for Claude-scored complexity metrics in the AAR experiments (categorizing research ideas, scoring code complexity)
