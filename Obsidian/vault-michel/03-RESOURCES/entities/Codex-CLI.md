# Codex CLI

Agente de codificação via linha de comando da OpenAI, concorrente direto ao Claude Code.

## O que é / What it is

Codex CLI é open-source e roda modelos GPT-4o/o1 no terminal com capacidades de leitura de codebase, execução de comandos e criação de PRs. Usa um arquivo de configuração equivalente ao `CLAUDE.md` do Claude Code. O modelo de permissão de ferramentas difere — menos granular que o sistema de allowlist do Claude Code.

## Relevância para o vault

Michel estuda Codex CLI como benchmark competitivo ao Claude Code. Comparar as abordagens de permissão, configuração e modelo de agente informa o design dos agentes e skills do vault.
