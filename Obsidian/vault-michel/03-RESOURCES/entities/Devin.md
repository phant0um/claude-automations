# Devin

Engenheiro de software autônomo da Cognition AI, primeiro produto posicionado como "AI software engineer".

## O que é / What it is

Devin é fully agentic: lê codebases inteiras, escreve Pull Requests, roda testes e itera até resolver issues autonomamente. Estabeleceu benchmarks no SWE-bench quando lançado em 2024. Posicionado acima do Claude Code em autonomia (menor supervisão humana requerida), com preço enterprise. A Cognition AI é uma startup focada exclusivamente em agentes de software.

## Relevância para o vault

Referência de teto de autonomia para agentes de software — Michel acompanha como ponto de comparação para o nível de autonomia desejável em seus próprios agentes de vault e ao avaliar novas ferramentas.
