# DuckDB

## Descrição

Database analítica in-process (sem servidor separado), columnar, vectorized. Alternativa SQLite para analytics. Rápido porque combina columnar storage + vectorized execution + zero-copy integration com Python/arrow. Útil para data analytics agents que processam datasets sem infraestrutura externa.
