# Flowise

Construtor low-code open-source para aplicações LLM com editor visual de fluxo drag-and-drop.

## O que é / What it is

Flowise permite montar pipelines RAG, chatbots e agentes sem escrever código. Self-hostable via Docker. Serve como alternativa visual ao LangChain code-first — mesmas abstrações (chains, retrieval, memory) mas acessíveis graficamente via nós conectados. Suporta múltiplos LLMs como backends.

## Relevância para o vault

Mencionado no vault como ferramenta para prototipagem rápida de workflows de IA. Alternativa para Michel experimentar pipelines de agente visualmente antes de portar a lógica para skills do vault em markdown/código.
