# GPT-5

Modelo frontier da OpenAI. Referenciado em research de meta-programming.

## Meta-programming behavior

Estudo arXiv: GPT-5.4 e Claude Opus 4.6, ao encontrar linguagem de programação desconhecida, não escrevem diretamente na linguagem alvo — escrevem programas Python que geram o código, depois debugam localmente. Esta estratégia de "meta-programming" é a key capability difference entre top agents e agents comuns.
