# GitHub

Plataforma de hospedagem Git com CI/CD, issues, PRs e GitHub Copilot, pertencente à Microsoft.

## O que é / What it is

GitHub é a maior plataforma de desenvolvimento colaborativo do mundo: repositórios git, GitHub Actions (CI/CD), Issues, Pull Requests e GitHub Copilot (AI code completion). O CLI `gh` permite acesso completo à plataforma via terminal. GitHub Codespaces oferece ambientes de desenvolvimento cloud sob demanda.

## Relevância para o vault

Michel usa GitHub para o histórico git do vault (`vault-michel`) e para projetos de código. O skill `commit-push-pr` automatiza o fluxo de commit/push/PR do vault via `gh` CLI.
