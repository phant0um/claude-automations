# Glean

Plataforma enterprise de busca com IA que indexa o conhecimento interno de organizações.

## O que é / What it is

Glean conecta-se a fontes como Slack, Google Docs, GitHub, Confluence e Jira para criar um índice unificado do conhecimento corporativo. Oferece busca semântica e respostas de IA sobre esse corpus privado. Integra com Claude enterprise search, permitindo que organizações usem Claude sobre dados privados via Glean como camada de acesso controlado.

## Relevância para o vault

Referência de arquitetura para o vault de Michel: Glean resolve para empresas o mesmo problema que o vault resolve individualmente — ter um segundo cérebro pesquisável com IA sobre conhecimento acumulado e privado.
