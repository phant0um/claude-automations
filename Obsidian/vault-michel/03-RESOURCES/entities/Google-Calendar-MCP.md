# Google Calendar MCP

Official Anthropic Connector. Send invites, decline events, generate meeting briefs, block focus time. Best combined with Gmail and Google Drive MCPs for full Google ecosystem access.

Setup: Claude → Customize → Connectors → Google Calendar.
