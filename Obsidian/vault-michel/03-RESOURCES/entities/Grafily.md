# Grafily

**GitHub:** github.com/TheBestTvarynka/grafily · por Pavlo Myroniuk (@TheBestTvarynka)

Plugin Obsidian para renderizar grafos familiares interativos a partir de arquivos Markdown (um arquivo por pessoa), usando ReactFlow com dois algoritmos de layout distintos.

## Contribuições relevantes

- Princípio: cada pessoa = uma nota `.md` com metadados estruturados (Spouse, Parents, Birth, Death, Image)
- Dois algoritmos: Reingold-Tilford (tree-based, centralização perfeita) e Brandes-Köpf (graph-based, universal)
- Motivação do autor: manter dados familiares privados sem depender de MyHeritage
- Não listado na store oficial — instalar via clone do repo + community plugins
