# IBM watsonx Assistant

Plataforma IBM Cloud p/ chatbots e assistentes virtuais. Renomeada de "Watson Assistant" p/ "watsonx Assistant" na linha watsonx (2023). Foco em IA empresarial com fallback p/ LLMs generativos.

## Características

- **Skills**: Dialog, Action, Search
- **Intents** (intenção do usuário) + **Entities** (slots)
- **Slot filling** automático (preenchimento de parâmetros)
- **Nova experiência (Actions)** — steps de conversação visuais
- **Integrações nativas**: Web Chat, Phone, Slack, Teams, WhatsApp, Facebook
- **Custom extension** via OpenAPI
- **Integração watsonx.ai** Studio + Runtime p/ IA Generativa

## Planos

| Plano | Uso |
|---|---|
| **Lite** | free, dev/teste |
| **Trial** | avaliação |
| **Plus** | produção, SLA |
| **Enterprise** | enterprise + SLA |
| **Enterprise w/ data isolation** | dados isolados |

## Conceitos

- **API Key** — autenticação
- **Assistant ID** — identifica o bot
- **System Prompt** — guia de IA Generativa
- **Session variables** — estado de conversa

## Concorrentes

- Amazon Lex (AWS)
- Microsoft Azure AI Bot Service
- Rasa (open-source)
- Manychat, Chatfuel (low-code)
