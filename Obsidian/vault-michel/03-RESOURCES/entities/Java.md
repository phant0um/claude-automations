# Java

Linguagem de programação orientada a objetos, compilada para bytecode executado na JVM. Fortemente tipada.

## Uso no FIAP ADS

Linguagem principal das Fases 5 e 6:
- **Fase 5**: POO, classes, encapsulamento, herança, polimorfismo
- **Fase 6**: Arrays, Collections, JDBC (conexão com banco de dados)

## Tópicos cobertos (FIAP)

- Estruturas de controle (`if`, `for`, `while`, `switch`)
- Classes e objetos
- Modificadores de acesso (`private`, `public`, `protected`)
- Getters e setters
- `extends`, `super`, `@Override`
- `ArrayList`, `HashMap`
- JDBC para Oracle DB
