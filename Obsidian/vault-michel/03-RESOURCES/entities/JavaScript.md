# JavaScript

Linguagem de programação web, única nativa dos browsers, com ecossistema Node.js para servidor.

## O que é / What it is

JavaScript roda no browser e no servidor (Node.js). TypeScript é o superset tipado que adiciona type safety e melhora o tooling. Conceitos fundamentais: async/await, Promises, fetch API e o event loop single-threaded. Frameworks principais: Vue, React, Angular para frontend; Express/Fastify para backend.
