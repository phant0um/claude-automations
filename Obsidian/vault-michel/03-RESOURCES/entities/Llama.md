# Llama

Família de LLMs open-source da Meta, referência em modelos locais e uso privado de IA.

## O que é / What it is

Llama (versões 3.1, 3.2, 3.3) tem pesos disponíveis publicamente sob licença permissiva. Roda localmente via Ollama com versões quantizadas (formato GGUF) que funcionam em laptops comuns. Llama 3.3 70B rivaliza com GPT-4o em muitos benchmarks e é a base da maioria das soluções de IA local do ecossistema open-source.

## Relevância para o vault

Chave para o conceito de `local-llm-privacy` no vault: Michel pode rodar modelos offline sem enviar dados para APIs externas, relevante para uso com material de concurso e conteúdo sensível da FIAP.
