# LlamaIndex

Empresa / projeto open-source focado em tooling para construção de aplicações LLM com acesso a dados. Desenvolve frameworks de RAG, conectores de dados e ferramentas de document processing para pipelines de AI agentivos.

## Produtos relevantes

- **LlamaIndex (framework):** framework principal para construção de pipelines RAG e indexação de documentos para LLMs.
- **LiteParse:** parser de PDFs e documentos office sem LLM, orientado a performance. Reescrito em Rust na v2.0 (2026-05-29), com targets Node, Python, WASM e Rust nativo. Speedup 5–100× vs. v1.
- **LlamaCloud:** plataforma cloud para document processing em escala.

## Posicionamento

LiteParse é posicionado explicitamente para **real-time agents**: "If you are running real-time agents and applications that need to read docs fast, LiteParse is the tool for it." Skills disponíveis para Claude Code via `npx skills add run-llama/llamaparse-agent-skills --skill liteparse`.
