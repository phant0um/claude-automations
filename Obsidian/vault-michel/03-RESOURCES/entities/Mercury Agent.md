# Mercury Agent

**Handle:** @mercury__agent (X/Twitter)

Sistema de agente autônomo com arquitetura de memória consciente/subconsciente, baseado em MemGPT, com capacidades de continuidade operacional de longa duração.

## Contribuições relevantes

- Arquitetura dual: memória consciente (hot — informação ativa atual) + memória subconsciente (archive — padrões e histórico, retrievado contextualmente)
- Baseado em MemGPT (Charles Packer et al.): LLMs como operating systems com gerenciamento de memória hierárquico
- Capabilities: memória persistente, execução permission-aware, git-native workflows, kanban orchestration, token-aware runtime
- Insight: context window longa ≠ memória — modelos lutam com informação no meio do contexto (Stanford "Lost in the Middle")
