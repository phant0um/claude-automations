# Meta

Empresa controladora do Facebook, líder em pesquisa de IA open-source via FAIR e modelos Llama.

## O que é / What it is

Meta opera Facebook, Instagram e WhatsApp. Em IA, a estratégia é radicalmente open-source: os modelos Llama são disponibilizados com pesos públicos, democratizando acesso a LLMs de ponta. FAIR (Fundamental AI Research) publica pesquisa de fronteira. PyTorch, o framework de ML mais usado em pesquisa, é mantido pela Meta. Hardware próprio de IA (MTIA) reduz dependência de NVIDIA.

## Relevância para o vault

A estratégia open-source da Meta é diretamente relevante para Michel: Llama via Ollama permite uso de IA sem custo de API e com privacidade total — chave para material sensível de concurso e estudo.
