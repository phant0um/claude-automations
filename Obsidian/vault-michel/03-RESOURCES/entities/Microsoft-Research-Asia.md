## Synthetic Computers at Scale (2026)

Tao Ge, Baolin Peng, Hao Cheng, Jianfeng Gao published a methodology for generating synthetic user computer environments at scale for long-horizon productivity agent training. 1,000 synthetic computers instantiated; each simulation runs >8h agent runtime and >2,000 turns. Occupation-specific skills extracted from simulations improve agent performance from 61.6% to 68.6% on held-out evaluations. Dataset released at HuggingFace (`microsoft/synthetic-computers-at-scale`). See [[03-RESOURCES/sources/ml-research-papers/synthetic-computers-at-scale-long-horizon-productivity]] and [[03-RESOURCES/concepts/agent-systems/synthetic-training-environments]].
