# Microsoft

Empresa de tecnologia com Azure cloud, GitHub, VS Code, Copilot e parceria estratégica com OpenAI.

## O que é / What it is

Microsoft opera Azure (um dos três grandes clouds), é owner do GitHub e criador do VS Code. O investimento de $10B+ na OpenAI e integração do GPT-4 no Copilot posicionou a empresa como líder em IA enterprise. Teams + Microsoft 365 Copilot levam IA diretamente para produtividade corporativa. Azure OpenAI Service hospeda modelos OpenAI em infraestrutura enterprise com compliance.

## Relevância para o vault

Michel usa diretamente dois produtos Microsoft no dia-a-dia: GitHub (histórico git do vault) e VS Code (desenvolvimento). O ecossistema Microsoft é inescapável no currículo ADS da FIAP.
