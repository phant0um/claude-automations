# Multica

Open-source platform for running and managing coding agents with reusable [[Agent Skills]].

## Overview

Enables structured agent orchestration with:
- Reusable skill system ([[Agent Skills Specification]])
- Agent lifecycle management
- Skill composition & chaining
- Cross-agent compatibility

## Context

Created by [[Forrest-Chang]], who also formalized [[Karpathy's Four Principles|Karpathy principles]] for [[03-RESOURCES/entities/Claude Code]]. Multica embeds these principles as guardrails for agent behavior.
