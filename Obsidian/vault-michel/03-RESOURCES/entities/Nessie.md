# Nessie

**Site:** nessielabs.com

"Context layer" para trabalho AI-native — ingere histórico de conversas de múltiplas ferramentas de IA (Claude, ChatGPT, Gemini, Perplexity, Cursor, Codex, Claude Code, Claude Cowork, OpenClaw) e vaults Obsidian, indexa numa "Library" pesquisável, expõe via CLI/MCP pra agentes consultarem contexto histórico em vez de re-explicar do zero a cada sessão.

## Relevância pro vault

Mesmo problema que [[03-RESOURCES/concepts/pkm-obsidian/knowledge-compounding|knowledge-compounding]] e [[04-SYSTEM/wiki/hot|hot cache]] tentam resolver internamente (fragmentação de contexto entre sessões/ferramentas) — Nessie é uma solução de terceiros pro mesmo problema, integrando direto com Obsidian.
