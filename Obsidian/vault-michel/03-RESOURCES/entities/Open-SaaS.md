# Open SaaS

Template SaaS open-source construído sobre o framework Wasp, com auth, pagamentos e banco de dados pré-configurados.

## O que é / What it is

Open SaaS é um boilerplate completo para produtos SaaS: autenticação (Auth.js), pagamentos (Stripe), banco de dados (Prisma + PostgreSQL) e deployment automatizado. O framework Wasp abstrai boilerplate full-stack com uma DSL declarativa que gera código React + Node.js. Reduz semanas de setup inicial a horas para um desenvolvedor solo.

## Relevância para o vault

Parte do `solo-saas-stack-2026` documentado no vault — Michel estuda esse stack como caminho para construir e lançar produtos SaaS como desenvolvedor solo com IA como co-piloto.
