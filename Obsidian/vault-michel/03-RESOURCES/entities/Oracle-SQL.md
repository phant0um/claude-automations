# Oracle SQL

Sistema de Gerenciamento de Banco de Dados Relacional (SGBDR) da Oracle Corporation. Banco de dados utilizado no curso FIAP ADS.

## Ferramentas associadas

- **Oracle SQL Developer** — IDE para administração e consultas
- **Oracle SQL Developer Data Modeler** — ferramenta de modelagem MER (`.dmd`)
- **Oracle XE** (Express Edition) — versão gratuita para desenvolvimento

## Uso no FIAP ADS

- **Fase 3**: Modelagem com Data Modeler, DDL básico
- **Fase 6**: SQL completo (DML, JOINs, GROUP BY), JDBC

## Diferenças do SQL padrão

- `ROWNUM` em vez de `LIMIT`
- `SYSDATE` para data atual
- `SEQUENCE` para auto-incremento (em vez de `AUTO_INCREMENT`)
- `VARCHAR2` em vez de `VARCHAR` (recomendado)
