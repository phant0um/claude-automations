# Prime Radiant

**Fundador:** Jesse Vincent (@obra)

Empresa de Jesse Vincent, criadores do framework Superpowers — sistema de skills composáveis para coding agents (Claude Code, Codex CLI, Gemini CLI e outros harnesses).

## Contribuições relevantes

- Criou o framework "Superpowers": skills composáveis que funcionam como job descriptions para agentes de código
- Skills são arquivos Markdown portáveis entre harnesses — instalados em `.claude/skills/` ou equivalente
- Filosofia: capacidades de agentes não devem ser acopladas a um harness específico
- Relacionado ao `npx skills@latest` CLI e ao ecossistema de skills para Claude Code
