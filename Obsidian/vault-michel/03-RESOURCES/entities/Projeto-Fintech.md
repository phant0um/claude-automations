# Projeto Fintech

Projeto prático central do curso ADS na [[02-AREAS/fiap/entities/FIAP|FIAP]]. Um sistema de gestão financeira pessoal desenvolvido progressivamente ao longo das 6 fases do primeiro ano.

## Fio condutor

Cada fase adiciona uma camada ao projeto:
- **Fase 1**: Concepção e primeiros algoritmos (Documento Visão)
- **Fase 2**: Prototipagem da interface (wireframes + Python)
- **Fase 3**: Modelagem do banco de dados (MER + DDL Oracle)
- **Fase 4**: Frontend web (HTML/CSS/JS)
- **Fase 5**: Backend orientado a objetos (Java)
- **Fase 6**: Integração com banco de dados (SQL + JDBC)
