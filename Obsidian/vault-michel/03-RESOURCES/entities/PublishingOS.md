# PublishingOS

**Origem:** 36 Publishing (36-publishing-claude-prompts.replit.app)

Biblioteca de 36 prompts estruturados para autopublicação com Claude, organizada como "sistema operacional" para o workflow completo de publicação de livros — desde pesquisa de nicho até escala de portfólio.

## Contribuições relevantes

- 36 prompts sequenciais cobrindo: market intelligence, production line, launch operations, scale architecture
- Framework de 5 partes para cada prompt: Role Assignment + Context Injection + Task Specification + Constraint Setting + Output Format
- Tese central: dominar 1 meta-skill (prompt engineering) substitui domínio de 12 disciplinas de publishing
- Insight de nicho: seleção de nicho é a decisão mais consequente — "mediocre book in hungry market wins"
