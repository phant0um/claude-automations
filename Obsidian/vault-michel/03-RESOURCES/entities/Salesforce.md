# Salesforce

Empresa de software enterprise (CRM, cloud, AI) com sede em San Francisco. Em 2026, tornou-se referência pública de transformação agêntica de engenharia em escala.

## Relevância para o vault

Salesforce é o principal caso documentado de adoção organizacional de Claude Code em engenharia enterprise, com métricas de produtividade publicadas e uma arquitetura de workflow agêntico replicável.

## Produtos e iniciativas AI

- **Agentforce** — plataforma de agentes AI para clientes Salesforce
- **Engineering 360** — plataforma interna que centraliza dados de centenas de sistemas (segurança, disponibilidade, qualidade, produtividade de dev)
- **AI Expert Suite** — suite interna de ferramentas AI para engineers
- **Salesforce Foundation Plugins** — biblioteca institucionalizada de Claude Code skills específicas para workflows Salesforce

## Transformação agêntica de engenharia (2026)

Sob liderança de [[03-RESOURCES/entities/Srinivas-Tallapragada]], Salesforce moveu de "90%+ de adoção de AI" para "agentic AI dirigindo o SDLC":

| Métrica | Resultado (Apr 2026 vs Apr 2025) |
|---|---|
| Work items por dev | +50.8% |
| PRs merged por dev | +79% |
| Effective Output Score (ML-based) | +151.3% |
| Total incidents | -5% (com 79% mais PRs) |
| Migração 33 endpoints | 231 person-days → 13 dias (18x) |

**Decisões chave:**
- Padronização em [[03-RESOURCES/entities/Claude Code]] como tool primária, rolled out para toda a organização
- Remoção de token limits (sinal cultural explícito)
- Claude Code skills como artefatos de engenharia compartilhados entre times
