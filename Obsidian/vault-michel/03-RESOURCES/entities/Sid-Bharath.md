# Sid Bharath

Desenvolvedor e blogger em sidbharath.com. Autor do artigo "How I Built Jarvis: A Personal AI Assistant With Claude Code" (Jan 21, 2026).

## O que fez

Construiu "Jarvis" — um life operating system pessoal completo rodando via Claude Code — sem escrever uma linha de código. O sistema integra Gmail, Google Calendar, task tracking, CRM pessoal, habit tracking e weekly reviews, tudo gerenciado por conversa em linguagem natural.
