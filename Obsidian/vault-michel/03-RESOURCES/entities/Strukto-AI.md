# Strukto-AI

Open-source organization behind [[03-RESOURCES/entities/AImirage]], a Unified Virtual Filesystem for AI agents.

- **GitHub:** github.com/strukto-ai
- **Primary product:** [[03-RESOURCES/entities/AImirage]] — VFS layer that mounts S3, Slack, GDrive, GitHub, MongoDB, SSH and more under a single Unix path tree
- **SDKs:** Python (`mirage-ai`), TypeScript (`@struktoai/mirage-node`, `@struktoai/mirage-browser`, `@struktoai/mirage-core`)
