# ViralBuilder

**Função:** Banco de dados de formatos de vídeo de anúncios de alto desempenho

Ferramenta de pesquisa de mercado para criação de anúncios de vídeo, usada no stack do Hermes Agent para identificar formatos vencedores de anúncios em plataformas de ecommerce antes de produzir o vídeo.

## Contribuições relevantes

- Provê banco de dados de formatos de vídeo de anúncios de alto desempenho por plataforma — substitui suposições por dados validados de mercado
- Similar ao Gethookd em função: pesquisa de criatives vencedores antes da produção
- Posição no stack kidpakerot: ViralBuilder (pesquisa) → Claude skill → Higgsfield → vídeo final
- Elimina o "guessing" sobre formato criativo ao fornecer evidência do que já converte
