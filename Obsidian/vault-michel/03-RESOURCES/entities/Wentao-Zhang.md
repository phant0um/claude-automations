# Wentao Zhang

Researcher and author of the Autogenesis paper (arXiv 2604.15034v1, April 2026).

## Work

- **Autogenesis: A Self-Evolving Agent Protocol** — introduced a framework for agents that autonomously improve their own operational protocols without external supervision or retraining.

## Key Contribution

Proposed the [[03-RESOURCES/concepts/agent-systems/self-evolving-agents]] paradigm: agents that run a monitor → identify → modify → evaluate loop on their own protocols, retaining only successful modifications.
