# awesome-llm-apps

Repositório: [github.com/Shubhamsaboo/awesome-llm-apps](https://github.com/Shubhamsaboo/awesome-llm-apps)
Criador: Shubham Saboo (@Saboo_Shubham_)
Licença: Apache-2.0
Tutoriais: [theunwindai.com](https://theunwindai.com/)

Cookbook de 100+ templates prontos para rodar de AI agents e RAG apps — clona, customiza, faz deploy em 3 comandos. Cobre o stack moderno completo: AI Agents, Multi-agent Teams, MCP Agents, Voice AI, RAG, Agent Skills, Fine-tuning.

## Diferenciais

- **Hand-built**: cada template é trabalho original, testado end-to-end antes de publicar
- **3 comandos para rodar**: sem `requirements.txt` quebrado, sem scaffolding incompleto
- **Provider-agnostic**: Claude, Gemini, GPT, Llama, Qwen, xAI via mudança de config
- **Apache-2.0**: fork, ship, sell — sem paywall, sem telemetria
