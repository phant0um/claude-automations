# Claude Models

Linha completa de modelos Claude da Anthropic disponíveis em junho de 2026.

## O que é / What it is

A linha Claude junho 2026 inclui: Haiku 4.5 (rápido, bulk tasks, mais barato), Sonnet 4.6 (padrão no Claude Code, balanceado), e Opus 4.7/4.8 (mais capaz, alto custo). Extended thinking está disponível em Sonnet 4.7+ e Opus. Cada modelo tem um ID específico para uso na API — ex.: `claude-sonnet-4-6`, `claude-haiku-4-5-20251001`.

## Relevância para o vault

Michel seleciona modelos conforme a tarefa: Sonnet para trabalho diário no vault, Opus quando Sonnet é insuficiente (raciocínio complexo, análise longa), Haiku para subagentes de baixo custo em pipelines paralelos.
