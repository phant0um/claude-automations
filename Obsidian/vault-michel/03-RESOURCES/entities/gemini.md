# Gemini

Família de LLMs do Google, distinguida por context window de 1M+ tokens e capacidades nativas multimodais.

## O que é / What it is

Gemini inclui modelos como 1.5 Pro, 2.0 Flash e Ultra, todos nativamente multimodais (texto, imagem, áudio, vídeo). A context window de 1M tokens é o maior diferencial competitivo frente ao Claude. Integra profundamente com Google Workspace (Docs, Drive, Gmail). O Gemini CLI é a resposta do Google ao Claude Code para uso no terminal.

## Relevância para o vault

Mencionado em fontes do vault como principal concorrente ao Claude em tarefas de contexto longo. Michel acompanha o espaço de CLI agents, e Gemini CLI representa o player Google nesse nicho.
