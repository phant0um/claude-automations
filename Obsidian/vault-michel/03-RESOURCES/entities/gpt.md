# GPT

Família de modelos da OpenAI, base do ChatGPT e referência do ecossistema de desenvolvimento de IA.

## O que é / What it is

GPT inclui GPT-4o (multimodal, padrão consumer), o1/o3 (reasoning models com chain-of-thought interno), e o4-mini (rápido/barato). O produto ChatGPT popularizou LLMs para o público geral. Codex foi o predecessor que alimentou o GitHub Copilot. Principal concorrente do Claude em casos de uso de desenvolvimento e análise.

## Relevância para o vault

Michel acompanha GPT como referência de mercado e para comparação de capacidades. Codex CLI (OpenAI) é o concorrente direto ao Claude Code que Michel estuda como benchmark.
