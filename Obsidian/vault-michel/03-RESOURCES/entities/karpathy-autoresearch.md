# karpathy/autoresearch

**GitHub:** github.com/karpathy/autoresearch · por Andrej Karpathy

Repositório original de Andrej Karpathy que define o padrão "autoresearch" — minimal autonomous loop onde um agente propõe mudanças, executa experimentos, avalia contra uma fitness function e mantém ou reverte com base nos resultados.

## Contribuições relevantes

- Padrão keep-or-revert: propõe hipótese → executa → avalia → mantém se melhorou, reverte se não
- Insight central: "If you can measure it, you can optimize it" — aplica-se a qualquer métrica mensurável
