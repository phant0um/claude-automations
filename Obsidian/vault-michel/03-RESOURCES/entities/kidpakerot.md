# kidpakerot

**Handle:** @kidpakerot (X/Twitter)

Criador do stack agentico Hermes + Claude + Higgsfield + ViralBuilder para produção automatizada de vídeos de anúncios para ecommerce.

## Contribuições relevantes

- Demonstrou stack de 4 ferramentas: ViralBuilder (pesquisa de formatos vencedores) → Claude skill `video-prompt-builder` → Higgsfield MCP (renderização) → orquestrado pelo Hermes
- A skill `video-prompt-builder` gera 4 seções: effect timeline, effect inventory, density map, energy arc
- Hermes como context layer: carrega brand context, armazena skills, gerencia routing rules
- Redução documentada: produção de anúncios de $500–$2.000 + 3 horas → 10 minutos a custo de assinatura
