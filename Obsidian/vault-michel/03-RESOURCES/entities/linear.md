# Linear

Ferramenta de gestão de projetos para times de software, com UX superior ao Jira e API bem documentada.

## O que é / What it is

Linear organiza trabalho em issues, cycles (sprints) e roadmaps com interface extremamente responsiva e atalhos de teclado abrangentes. API bem documentada permite integrações programáticas. Adotado por startups de produto como alternativa mais leve ao Jira (Atlassian). MCP server para Linear existe para integração com Claude Code.

## Relevância para o vault

Mencionado no vault como possível sistema de tracking de projetos de Michel. A integração via MCP (disponível como plugin) permitiria gerenciar issues de projetos diretamente de dentro do Claude Code.
