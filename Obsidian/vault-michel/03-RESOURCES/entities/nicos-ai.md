# nicos-ai

**Handle:** @nicos_ai (X/Twitter)

Engenheiro de software sênior que documenta workflows de alta performance com Claude Code. Articula a mudança de paradigma da "nova habilidade de engenharia": dirigir agentes, não digitar código.

## Contribuições relevantes

- **Workflow de 5 agentes:** brainstorming → design técnico → implementação → revisão → validação
- **Git worktrees como multiplicador:** 4–8 sessões paralelas de Claude Code em branches separadas
- Insight: *"A nova habilidade de engenharia não é só saber codificar. É saber dirigir vários agentes sem perder o controle do sistema."*
