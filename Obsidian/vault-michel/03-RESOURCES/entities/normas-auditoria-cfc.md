## Clusters para Estudo (Concurso)

1. **Fundamentos** → NBC TA 200 + Estrutura Conceitual (ler primeiro)
2. **Planejamento/Risco** → NBC TA 300 → 315 → 320 → 330 (sequência lógica)
3. **Evidências** → NBC TA 500 (hub) → 501–580 (especializações)
4. **Relatório** → NBC TA 700 (hub) → 701, 705, 706, 710, 720
5. **Qualidade profissional** → NBC PA 01/02 → 11, 13, 400 + PO 900
6. **Armadilhas de prova** → NBC TA 220 (R3, vigência 2023) · NBC TA 315 (R2) · RES 1495 (CNAI)
