# obra — Jesse Vincent

Desenvolvedor open-source e fundador da [Prime Radiant](https://primeradiant.com). Criador do framework **Superpowers** — metodologia completa de desenvolvimento de software para coding agents.

- **GitHub:** https://github.com/obra
- **Blog:** https://blog.fsck.com
- **Sponsor:** https://github.com/sponsors/obra
- **Org:** [[03-RESOURCES/entities/Prime-Radiant]] (Prime Radiant)

## Superpowers

O projeto principal. Lançado em outubro de 2025 ([anúncio original](https://blog.fsck.com/2025/10/09/superpowers/)), superpowers é uma coleção de skills composáveis que transformam qualquer coding agent em um desenvolvedor que segue TDD, YAGNI, DRY e subagent-driven-development de forma automática — sem configuração manual por projeto.

Suporta: Claude Code, Codex CLI, Codex App, Factory Droid, Gemini CLI, OpenCode, Cursor, GitHub Copilot CLI.

Ver [[03-RESOURCES/entities/superpowers]] para detalhe do framework e [[03-RESOURCES/sources/claude-code-skills/obrasuperpowers-an-agentic-skills-framework-software-development-methodolog]] para análise completa.

## Filosofia

> "Your coding agent just has Superpowers."

Jesse acredita em metodologia como código: ao invés de explicar o processo para cada sessão, encodifica-se o processo em skills que o agent aplica automaticamente. Ênfase em TDD estrito, YAGNI, e subagent-driven-development como forma de autonomia sustentável.
