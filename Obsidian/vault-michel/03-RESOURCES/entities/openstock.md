# OpenStock

**GitHub:** github.com/Open-Dev-Society/OpenStock · AGPL-3.0

Alternativa open-source a plataformas pagas de mercado financeiro, construída pela Open Dev Society (ODS) — rastreia preços em tempo real, alertas personalizados e análise de empresas.

## Contribuições relevantes

- Stack: Next.js 15 + React 19 + TypeScript + MongoDB + Finnhub API + TradingView widgets
- AI opcional: Google Gemini (default), MiniMax, Siray — configurável via `AI_PROVIDER`
- Features: busca global (Cmd+K), watchlist por usuário, candlestick charts, sentimento (Reddit/X/news), daily news digest via cron
- Email AI: welcome personalizado por Gemini via Inngest; daily summary por watchlist
- Tagline: "Built openly, for everyone, forever free."
