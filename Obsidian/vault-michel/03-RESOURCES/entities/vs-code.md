# VS Code

Editor de código open-source da Microsoft, o mais popular do mundo para desenvolvimento.

## O que é / What it is

VS Code combina leveza com um ecossistema massivo de extensões. Suporta remote development (SSH, containers, Codespaces), settings sync entre máquinas e políticas enterprise. A extensão Claude Code integra o assistente diretamente no editor. GitHub Copilot é a integração AI nativa disponível via marketplace.

## Relevância para o vault

Michel usa VS Code como editor principal para código nos projetos FIAP e desenvolvimentos paralelos. A extensão Claude Code torna o vault acessível sem sair do editor, combinando IDE e segundo cérebro.
