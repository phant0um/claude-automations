# walkinglabs

Open-source organization behind the `learn-harness-engineering` GitHub repository — a multilingual beginner course on building reliable environments for AI coding agents.

## Notable Projects

- **learn-harness-engineering** — project-based course: 12 lectures, 6 projects, resource library (AGENTS.md, init.sh, feature_list.json templates). Available in EN, ZH, RU, VI, KO.
- **awesome-harness-engineering** — curated reference list for the field.

## Key Contribution

Synthesized and operationalized official harness engineering guidance from OpenAI (Codex) and Anthropic into a structured beginner curriculum. The course's `skills/harness-creator/` skill can scaffold a production-grade harness in minutes.
