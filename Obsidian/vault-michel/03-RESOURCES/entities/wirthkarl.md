# @wirthkarl (Karl Wirth)

Builder de systems agenticos e co-criador do **Nimbalyst** — workspace open-source para montar harness de coding agents. Autor do framework de 8 pilares para harness de equipes de desenvolvimento com IA.

## Contribuições

- **8-Pillar Harness Framework:** framework derivado de colapsar todos os failure modes recorrentes em pilares que cada um nomeia exatamente um problema. Pilares: Context, Provenance, Capability, Workflow, Restraint, Verification, Visual Interface, Coordination.
- **Nimbalyst:** workspace open-source para montar e operar harness de coding agents nos 8 pilares. Permite rodar Claude Code, Codex e outros agentes side by side sem reconstruir o layer acima.
