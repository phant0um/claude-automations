## Tese central
AI Agent problem: não é que não sabem responder — é que todo dia agem como te conheceram agora. Bug fixed ontem, esquece hoje. Preferences repetidas, esquece próxima round. Decisions, pitfalls, processes trapped em single conversation/directory. Solução: long-term memory layer.

## Key insights
- "每次都像刚认识你" — every time like just met you
- Decisions/pitfalls/processes trapped em single session
- Long-term memory layer = solution para amnesia entre sessions
