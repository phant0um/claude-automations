## Tese central
Agente IA = while-loop: LLM seleciona ação → executa → avalia → repete até task pronta. 6 partes: cérebro, planejamento, ferramentas, memória, loop, guardrails.

## Key insights
- Brain: chatbot escreve texto; agent TOMA DECISÕES. Planning: CoT/ToT/Reflexion transformam objetivo vago em ações claras.
- Tools = funções (MCP comum); Memory = curto (context window) + longo (vector store, files, KB). Window cheio → resumo + adiante.
- Guardrails (sandbox, human checks, token limits, scope) impedem autonomia → caos caro. Mais autonomia, mais importam.
