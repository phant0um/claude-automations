## Tese central
Hermes Agent UI transforma agente terminal-only em dashboard com chats, files, profiles, swarms, boards, missions, workflows.

## Key insights
- Dois paths: WebUI (simples, 1 agente, baixa fricção) e Workspace (swarms, kanban, conductor, mobile, customização).
- Terminal escala mal além de 1 comando; UI = camada visual sem remover poder do agente.
- Workspace = setup avançado; WebUI = entrada amigável pra quem não quer 4 terminais abertos.
