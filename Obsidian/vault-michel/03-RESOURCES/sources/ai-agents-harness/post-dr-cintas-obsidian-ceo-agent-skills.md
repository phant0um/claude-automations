# Obsidian Skills — kepano (Steph Ango, CEO Obsidian)

> Título original provavelmente continha "kepano" (CEO Obsidian) — não "Dr Cintas" (sem match encontrado; possível erro de OCR/transcrição do nome de autor).

## Resumo

Steph Ango (kepano), CEO da Obsidian, lançou **obsidian-skills**: coleção oficial de 5 Agent Skills que ensinam Claude Code (e outros agentes) a trabalhar com formatos Obsidian.

**5 skills:**
1. **obsidian-markdown** — bridge entre formatos Obsidian e agentes de código
2. **obsidian-bases** — ensina formato `.base` (databases dentro do vault: table views, queries filtradas, registros linkados)
3. **json-canvas** — formato aberto de canvas infinito (mind maps, flowcharts, diagramas de arquitetura)
4. **obsidian-cli** — acesso via linha de comando ao vault
5. **defuddle** — limpeza de páginas web ruidosas pra pesquisa/resumo dentro do vault

## Por que importa (vault)

Este vault já usa skills equivalentes: `obsidian-cli`, `obsidian-bases`, `obsidian-markdown`, `json-canvas`, `defuddle` (ver lista de skills disponíveis) — confirma alinhamento com padrão oficial kepano. Ver entity [[03-RESOURCES/entities/kepano]].
