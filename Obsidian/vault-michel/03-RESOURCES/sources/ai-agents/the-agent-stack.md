## Tese central
Camadas do agent stack: model → tools → orchestration → memory → eval. Entender o stack é prerequisite para arquitetar systems.

## Key insights
- Stack layers: model, tools, orchestration, memory, eval
- Cada layer tem suas trade-offs e patterns
- Full-stack agent engineering requer entender todas as camadas
