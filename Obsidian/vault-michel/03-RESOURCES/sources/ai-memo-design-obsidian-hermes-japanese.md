## Tese central
Diferença em notas para AI não é texto volume — é estrutura: summary, source, context, links, next actions separados para AI não se perder. Obsidian + Hermes Agent/Claude Code nota design.

## Key insights
- AI-readable notes: summary + source + context + links + next actions separados
- "AIが迷わない形" = formato que previne AI de se confundir
- Estrutura > volume
