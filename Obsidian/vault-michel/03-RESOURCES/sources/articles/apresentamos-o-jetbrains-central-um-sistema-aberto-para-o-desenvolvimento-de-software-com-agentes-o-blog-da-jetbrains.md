## Tese central

JetBrains Central é sistema aberto para desenvolvimento de software com agents — integra agents de IA no chat do JetBrains AI, conecta com ferramentas externas via padrões abertos.

## Key insights

- Open system: não walled garden, integra com ecosystem agentic
- Agent development como primeira classe no IDE
- JetBrains entrando no agent orchestration layer
