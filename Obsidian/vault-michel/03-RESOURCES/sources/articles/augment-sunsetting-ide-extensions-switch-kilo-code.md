# Is Augment sunsetting its IDE extensions?

Augment users reportaram sunset notice de 1 mês para IDE extensions. Augment está movendo upmarket para enterprise + Cosmos agent platform. Kilo Code posiciona-se como alternativa para JetBrains users que querem autocomplete + agent panel in-IDE.

## Tese Central

Augment está shutdown seus IDE extensions e movendo para enterprise/Cosmos — deixando individual devs e small teams em JetBrains sem tooling. Kilo Code oferece alternativa: plugin JetBrains nativo (Version 7 rebuilt from ground up), open-source (Apache-2.0), pricing transparente (500+ models, no markup com BYOK), sem $100 floor ou expiring credit pool.

## Pontos-Chave

### O que Aconteceu

- Março 2026: Augment removeu Next Edit e Completions em non-enterprise plans
- Agora: sunset completo dos IDE extensions (~1 mês notice)
- Alternativa da Augment: Augie CLI + Cosmos (workflows muito diferentes, CLI clunky, sem GUI)
- Pricing: Business plan $100/mês até 50 seats; Community plan gone; subreddit restricted, Discord closed
- **Por que:** Augment não vai bankrupt — move upmarket para enterprise teams e Cosmos

### Kilo Code JetBrains Plugin

- Inline Tab autocomplete + Next Edit mode (multi-line predictions)
- Agent panel dentro de IntelliJ, PyCharm, GoLand, etc.
- Checkpoint tracker: ver e rollback mudanças
- Version 7: rebuilt as native JetBrains plugin (não WebView), Kilo CLI runtime bundled, sign-in/model selection/permissions nativos
- Kiril (ex-JetBrains, ~10 anos) no time
- Stable no Marketplace, todas JetBrains IDEs (Community + Ultimate)

### Open Source, Open Pricing, Open Models

- Apache-2.0
- 500+ models via gateway, no markup com BYOK
- Modelo visível em cada request, context window size, full prompt
- Sem $100 floor, sem expiring credit pool
