# DBeaver Announces Native Integration with Databricks

DBeaver Corporation e Databricks anunciam parceria expandida com launch de native Databricks integration — advanced SQL development e data exploration capabilities. Joint customers trabalham efficientemente através de Databricks workflow enquanto mantêm acesso a 100+ databases em interface unificada.

## Tese Central

Data teams lutam com fragmented tools e inconsistent SQL development experiences across data sources. A integração nativa DBeaver-Databricks fornece complete visibility em data processes (raw ingestion → production-ready datasets), com native Databricks authentication (user-to-machine e machine-to-machine) e seamless cloud integration (AWS, GCP, Azure).

## Pontos-Chave

### Key Capabilities

- **Advanced SQL Editor**: smart autocomplete semântico, code formatting, debugging, AI-powered features (AI Chat, AI Code Completion, AI Error Fix)
- **Built-in Visualizations**: datas, numbers, images, spatial data em interactive maps, quick charting
- **Seamless Integration**: native Databricks auth (U2M e M2M), AWS/GCP/Azure
- **Tableau Integration**: transform data em reports e dashboards
- **Multi-Database Support**: Databricks + 100+ SQL/NoSQL databases em interface única

### Disponibilidade

- Databricks connectivity em DBeaver Community e todos DBeaver PRO editions
- Features avançadas (enterprise auth, visualization) exclusivas DBeaver PRO
