# The Complete Guide to Claude Plugins (10x your productivity)

## Tese central

Plugins no Claude Cowork são a camada de automação mais poderosa disponível: combinam Connectors (serviços externos) + Skills (workflows reutilizáveis) em um único "papel" que sabe como o usuário trabalha. 28 plugins oficiais da Anthropic já estão disponíveis; o ecossistema de terceiros está crescendo.

## Key insights

- **Hierarquia de extensão Claude:**
  - **Connectors** → integra serviços externos (Gmail, Calendar, Figma, Slack, Notion)
  - **Skills** → workflows reutilizáveis salvos
  - **Plugins** → Connectors + Skills em um pacote completo com "papel" (role) definido
- **Acesso:** Claude desktop → Cowork → Customize → Plugins
- **28+ plugins oficiais Anthropic** cobrindo Legal, Marketing, Engineering, Finance, Design, Productivity
- **Marketplace de terceiros:** claudemarketplaces.com, aitmpl.com/plugins, github.com/anthropics/claude-code/tree/main/plugins
- **Criar do zero:** prompt Cowork sobre o que construir, ou upload de arquivo/GitHub link
- **Scheduled Tasks:** rodar plugins em schedule (ex.: daily morning brief automático)
- **Fluxo de exemplo:** "Read emails last 24h + calendar + Notion high-priority tasks → morning briefing" — um prompt, 3 ferramentas, < 1 min
- **Pro tip de segurança:** em vez de baixar plugin de terceiro, peça ao Cowork para recriá-lo — sem risco de malware
- **Import existing skills → Cowork builds plugin:** caminho mais fácil para primeiro plugin
