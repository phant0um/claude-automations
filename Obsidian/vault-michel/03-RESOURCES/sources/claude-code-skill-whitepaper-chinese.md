## Tese central
Pare de repetir ensino para AI. Um Markdown file transforma sua experiência em Claude reusable work capability. Skills resolvem: PR description format, code review risk-first, doc conventions — tudo codificado em vez de re-stated.

## Key insights
- Skill = Markdown file que codifica experience
- Sem skill: mesma instrução repetida todo session
- Skill = reusable work capability
