# Top 10 Agent Skills by GitHub Stars

## Tese Central

The agent skills market shows clear signal: small, sharp workflows are winning. The top 10 most-starred GitHub agent skills reveal what the community values in composability.

---

## Conteudo Original

![Image](https://pbs.twimg.com/media/HK44q0HWcAAm1cg?format=jpg&name=large)

The agent skills market has a clear signal: Small, sharp workflows are winning. I refreshed the top 10 most-starred agent skills on GitHub:

1. 228,740 stars Agentic skills framework and dev methodology. [https://github.com/obra/superpowers](https://github.com/obra/superpowers)
2. 151,088 stars Official public Agent Skills repo from Anthropic. [https://github.com/anthropics/skills](https://github.com/anthropics/skills)
3. 130,016 stars Real-world skill setup from Matt Pocock. [https://github.com/mattpocock/skills](https://github.com/mattpocock/skills)
4. 110,407 stars Claude Code setup for exec, design, engineering, docs, QA. [https://github.com/garrytan/gstack](https://github.com/garrytan/gstack)
5. 92,040 stars Design intelligence for better UI/UX. [https://github.com/nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)
6. 60,442 stars Turns code into an interactive knowledge graph. [https://github.com/Egonex-AI/Understand-Anything](https://github.com/Egonex-AI/Understand-Anything)
7. 60,265 stars Production-grade engineering skills for coding agents. [https://github.com/addyosmani/agent-skills](https://github.com/addyosmani/agent-skills)
8. 53,903 stars AI-powered job search system using Claude Code skill modes. [https://github.com/santifer/career-ops](https://github.com/santifer/career-ops)
9. 44,469 stars Taste skill that pushes agents away from generic output. [https://github.com/Leonxlnx/taste-skill](https://github.com/Leonxlnx/taste-skill)
10. 42,815 stars Researches trends across Reddit, X, YouTube, HN, and the web. [https://github.com/mvanhorn/last30days-skill](https://github.com/mvanhorn/last30days-skill)

The interesting part: 6 of the top 10 are single-skill repos. Not huge libraries. One sharp capability.
