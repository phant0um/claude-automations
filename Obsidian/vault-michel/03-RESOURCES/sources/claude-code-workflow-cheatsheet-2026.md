## Conteúdo Disponível

### Getting Started

Instalação do Claude Code (requer Node.js 18+):

```bash
curl -fsSL https://claude.ai/install.sh | bash
cd seu-projeto
claude
/init
```

O `/init` gera o `CLAUDE.md` inicial do projeto — ponto de entrada para toda a configuração.
