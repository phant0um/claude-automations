# Claude Design Masterclass for Beginners

**Autor:** [[03-RESOURCES/entities/Corey-Ganim]] (@coreyganim)
**Tipo:** Guia/masterclass prático
**Ferramenta:** [[03-RESOURCES/concepts/claude-design]] — disponível em claude.ai/design para usuários Claude Pro/Max

## O que é Claude Design

Claude Design não é Canva nem Figma. É uma ferramenta de design conversacional powered by **Claude Opus 4.7**, acessível via navegador. Interface de dois painéis: chat (esquerda) + canvas de design (direita).

**Diferenciais:**
- **Lê sua marca automaticamente** — importa codebase e arquivos de design ao criar um projeto; herda cores, tipografia e padrões de componentes sem configuração manual
- **Entende contexto** — aceita imagens de referência, documentos, e captura elementos diretamente do site via web capture tool
- **Exporta para qualquer formato** — PDF, PPTX, HTML standalone, Canva, ou link interno para colaboração
- **Handoff para código** — empacota tudo em "handoff bundle" para passar ao Claude Code com uma instrução

## Workflow Central

### 1. Criar Projeto (Brand Integration Automático)
- Clicar "New Project" em claude.ai/design
- Sistema puxa automaticamente o design system da organização
- Não é necessário reconfigurar marca a cada projeto

### 2. Dar Contexto ao Claude
- Imagens de referência, brand guidelines (PDF), copy/headlines, audiência, constraints de estilo
- Regra: quanto mais específico, menos revisões

### 3. Escrever o Prompt (sem precisar de linguagem de design)
Fórmula eficaz:
- **Goal:** o que está sendo construído (landing page, pitch deck, social graphic)
- **Layout:** como organizar (hero + features, três colunas)
- **Content:** informações incluídas (headlines, body, CTAs)
- **Audience:** para quem é
- **Tone:** sentimento (professional, minimalist, luxury, playful)

### 4. Iterar

| Método | Uso |
|--------|-----|
| Chat | Mudanças estruturais ou estéticas grandes |
| Inline Comments | Tweaks em elemento específico (clicar → instruir) |
| Direct Editing | Editar texto direto no canvas |
| Adjustment Knobs | Sliders custom criados pelo Claude (spacing, color intensity, typography weight) |

### 5. Exportar
- Internal URL, PDF, PPTX, HTML, Canva, Handoff bundle para Claude Code

## Modelo de Negócio / ROI

- **Preço:** incluso no Claude Pro ($20/mês) ou Max ($200/mês)
- **ROI solopreneur típico:** 4 landing pages + 2 pitch decks + 8 campanhas/mês → economia de $8K–$15K em design ou 60+ horas de labor

---

## Limitações (research preview)

1. Animações complexas não suportadas
2. Renderização 3D limitada (formas básicas funcionam)
3. Sem integração com dados ao vivo (design estático)
4. Não pixel-perfect como Figma (95% mas não 100%)
5. Projetos muito complexos podem precisar ser divididos
6. Colaboração turn-based, não simultânea (≠ Google Docs)

---

## Quando Usar vs Alternativas

| Ferramenta | Melhor para |
|-----------|-------------|
| **Claude Design** | Rapidez, budget limitado, não-designer, branded consistency, iteração rápida |
| **Figma** | Pixel-perfect, sistemas de design complexos, times de design, handoff profissional |
| **Canva** | Templates simples, social media, sem curva de aprendizado |
| **Designer contratado** | Brand identity, rebrand anual, work core do produto, direção criativa |

## Fórmula de Prompt Efetiva

```
Goal: [o que está construindo]
Layout: [como organizar]
Content: [informações que entram]
Audience: [para quem é]
Tone: [que sentimento transmitir]
```

**Exemplo:**
> "Create a landing page for a productivity tool. The audience is busy executives who value simplicity. Make it clean, professional, and minimalist. Include a hero section with headline, subheadline, and CTA button. Below that, three feature cards with icons."

---

## Bottom Line

Claude Design não substitui designers sênior em trabalho complexo de brand. Para os 80% de trabalho visual que solopreneurs e pequenas empresas precisam (landing pages, pitch decks, social graphics, protótipos) — é genuinamente transformador. Remove a fricção entre "tenho uma ideia" e "tenho algo polished".
