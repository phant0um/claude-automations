## Tese central

Roadmap técnico de 16 passos para construir um trading agent que opera em Polymarket (prediction markets). Harness engineering: a infraestrutura que conecta agent ao mercado, gerencia risk, e execution.

## Key insights

- Trading agent = agent + harness (market connection, risk management, execution layer)
- 16 passos estruturados — não é vibe coding, é engineering
- Polymarket como case study: prediction markets têm estrutura clara para agent trading
