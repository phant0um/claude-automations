## Tese Central

Research do IC3 (Initiative for CryptoCurrencies and Contracts) encontra que a maioria das plataformas AI-agent crypto não mostra evidência clara de execução autônoma de trades, apesar de valuations combinadas de tokens excedendo $3 bilhões. Token holders perderam $191.7M coletivamente enquanto agent treasuries retêm $30M+ em paper gains.

## Pontos-Chave

1. **Scope**: 1,900+ AI-tagged crypto projects surveyed, narrowed to investment-focused agents, 10 representative platforms analyzed. On-chain analysis cobriu 11 Solana-based agent treasuries com 925,323 token holders.
2. **Finding central**: muitas plataformas não providem evidência clara de autonomous trade execution. Developer interviews sugerem que deployments visíveis são basic API integrations — software que relay user instructions to exchanges, não gerando e executando trades independentemente.
3. **Market-cap-to-AUM ratios**: exceeded 10,000x across platforms, vs below 1x for established DeFi protocols.
4. **Distribution de returns**: agent treasuries retêm $30M+ em paper gains. Token holders como classe perderam $191.7M. Top 1% de wallets capturou 81.4% de todos os gains ($1.81B). Tokens declinaram 93% em média do ATH.
5. **3 critérios de maturidade**: autonomous execution, risk-adjusted profitability, stakeholder alignment. Nenhuma das 10 plataformas examinadas meeting all three.
6. **Frameworks analisados**: ElizaOS (open-source AI-agent runtime, ai16z) e Virtuals Protocol (tokenize autonomous agents). Finding "no clear evidence" aplica-se genericamente ao broader sample, não a estes especificamente.

## Conceitos

- Autonomous trade execution como critério de maturidade
- Market-cap-to-AUM ratio como signal de speculative bubble (10,000x vs <1x)
- Agent treasury vs token holder return divergence
- Stakeholder alignment como critério
