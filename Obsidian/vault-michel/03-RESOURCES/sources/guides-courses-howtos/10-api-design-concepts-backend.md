## Tese central
10 conceitos fundamentais de design de API para backend engineer: do básico ao avançado.

## Key insights
- Lista: Idempotência, Paginação (cursor/offset), Rate Limiting, Versionamento, Filtering/Sorting, Timeouts/Retries.
- Avançado: HATEOAS, API Gateway, Partial Responses (field selection), Error Modeling consistente.
- Padrão emergente: cursor pagination > offset pra escala; idempotency key obrigatório em writes.
