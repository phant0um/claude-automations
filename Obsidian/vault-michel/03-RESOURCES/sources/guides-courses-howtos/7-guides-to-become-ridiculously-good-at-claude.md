## Tese central

Um curated learning path em 7 guias ordenados para dominar Claude: do básico absoluto até vibe-coding com Claude Code, passando por Cowork, Skills, e treinamento de voz pessoal — com a sequência correta de leitura e tempo estimado por guia.

## Argumentos principais

- A maioria dos guias existentes do autor está desatualizada — este post mapeia o caminho atual correto
- Ordem importa: fundamentos → features específicas (Cowork → Skills) → customização avançada (voz, certificação, anti-AI-talk)
- Vibe-coding (Claude Code sem escrever código) é o skill faltante que complementa todos os outros
- Cowork é o melhor recurso para quem não programa; Skills é o melhor para tarefas recorrentes

## Key insights

- Guia 1 (Claude For Dummies): ~5 min, para nunca-usou Claude
- Guia 2 (Claude basics): ~9-10 min, o que se pode fazer — nota: modelo mais atual é 4.7 não 4.6
- Guia 3 (Claude Cowork): ~20 min — melhor newsletter do autor, o melhor feature para não-programadores
- Guia 4 (Claude Skills): ~15 min + prática contínua criando novas skills — melhor para colaboração (skills são compartilháveis)
- Guia 5 (Sounding like you): ~40 min — 100 perguntas de entrevista para treinar Claude na sua voz; mais viral no Substack
- Guia 6 (Claude Certified): ~3 horas total (1h por certificado, 3 certificados) — certificação gratuita para LinkedIn
- Guia 7 (Avoid AI-talk): ~7 min — como não soar como IA
- Vibe-coding demo: 8 passos para criar um chatbot com knowledge das newsletters em ~14 minutos usando GitHub + Vercel sem escrever código

## Exemplos e evidências

- Demo ao vivo: chatbot "Chat with Ruben, about Claude" em [chat-with-ruben.vercel.app](https://chat-with-ruben.vercel.app)
- Setup de vibe-coding: bypass permissions mode → conectar GitHub + Vercel → upload arquivos de contexto → screenshot de referência → prompt com /goal
- 84.000+ leitores da newsletter do autor
- claude101.com: site gratuito vibecoded com os melhores guias

## Implicações para o vault

Referência de learning path para usuários Claude. O conceito de "vibe-coding" é aplicável ao desenvolvimento de tools e scripts do vault sem escrever código manualmente. A ênfase em Skills como veículo de colaboração (compartilháveis) reforça o design de skills deste vault para serem portáteis.
