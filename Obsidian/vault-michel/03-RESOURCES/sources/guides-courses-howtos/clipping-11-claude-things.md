# 11 Claude Things I Wish Someone Told Me

**@InduTripat82427**: shift de mindset — chatbot → cognitive partner → system builder.

## 11 Patterns

1. **Projects** são subestimados — dump context uma vez, pare de re-explicar. Salva 100+ horas.
2. **CLAUDE.md matters more than prompts** — operating manual para o AI. Define quem você é, o que faz.
3. **Conversation limits** — sessões longas degradam. Fresh context para cada task nova.
4. **Extended Thinking** — ative para problemas complexos. Diferença entre resposta rápida e análise profunda.
5. **Projects tem versioning** — histórico de instruções, rollback disponível.
6. **Artifacts** — Claude constrói apps funcionais dentro do chat (calculadoras, dashboards, jogos).
7. **Haiku para tarefas simples** — 30x mais barato que Sonnet para respostas triviais.
8. **Uploads além de PDFs** — CSV, código, imagens, audio (experimental).
9. **Share conversation** — link permanente, útil para handoffs.
10. **Claude Code hooks** — automações determinísticas que disparam em eventos específicos.
11. **Multi-file edit** — Claude edita múltiplos arquivos em uma operação, mantendo consistência.
