# 20 AI Concepts You Must Understand in 2026

**@sairahul1**: 20 conceitos fundamentais para entender como AI realmente funciona. Sem PhD, sem jargão — mental models visuais.

## Estrutura (4 partes)

**Parte 1 — Como AI Funciona (fundação):**
1. Neural Networks — pipeline de camadas, weights, training = ajustar billions de weights
2. Transformers — attention mechanism, parallel processing de tokens
3. Embeddings — representação vetorial de significado semântico
4. Tokenization — texto → tokens (não palavras, não caracteres)
5. Inference — como o modelo gera output

**Parte 2 — LLMs:**
6. Pretraining vs Fine-tuning
7. Context Window — limite de atenção por sessão
8. Temperature / Sampling — como o modelo "escolhe" o próximo token
9. Hallucination — por que modelos inventam
10. System Prompt — instrução de contexto invisível ao user

**Parte 3 — Agentes e RAG:**
11. RAG (Retrieval-Augmented Generation) — grounding em dados externos
12. AI Agents — observation → decision → action loop
13. Tool Use / Function Calling — agent como orchestrator de tools
14. Prompt Engineering vs Context Engineering

**Parte 4 — Treinamento:**
15. RLHF — Reinforcement Learning from Human Feedback
16. RLAIF — RLHF com AI como judge
17. Fine-tuning — SFT, DPO, GRPO
18. Distillation — modelo menor aprende de modelo maior
19. Quantization — reduzir precisão para reduzir VRAM
20. Multimodal — texto + imagem + audio + vídeo
