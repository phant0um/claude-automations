# 25 Ways to Use Claude: Complete Interface & Tool Guide

Comprehensive overview of Claude interfaces and when to use each.

## Core Interfaces (1-6)

1. **Chat** → Quick questions/answers (don't use Cowork for "write an email")
2. **Opus 4.7** → Deep thinking, complex writing, professional work
3. **Sonnet 4.6** → Speed + quality balance (grammar, summaries, short responses)
4. **Extended Thinking** → Difficult problems (enable/disable based on complexity)
5. **Cowork** → Serious work (.docx, .xlsx, .pptx) — "functions like an employee"
6. **Claude Code** → Developers + curious builders (terminal, web/app dev)

## Specialized Tools (7-13)

7. **Claude in Excel** → Real formulas, models, templates (add-in from store)
8. **Gamma Connector** → Presentation generation from prompts (with research)
9. **Claude Design** → Website mockups/prototypes (claude.ai/design, 1 hour)
10. **Chrome Extension** → Browser agent (reads pages, acts within browser)
11. **Computer Use** → Mac automation (click, type, navigate)
12. **Dispatch** → Remote execution (text task from phone, runs on computer)
13. **Web Search** → Live web research before responding

## Advanced Modes (14-25)

14. **Vision** → Image analysis
15. **Projects** → Persistent context/knowledge base
16. **Research Mode** → Extended analysis
17. **Artifacts** → Shareable code/design outputs
18. **Connectors** → Third-party integrations (Slack, Notion, etc.)
19. **Custom Instructions** → Session-level behavior
20. **Files** → Document processing
21. **Citations** → Sourced responses
22. **Thinking** → Deep reasoning
23. **Memory** → Session-persistent learning
24. **APIs** → Direct integration
25. **Batch Processing** → Bulk operations

## Decision Tree (When to Use What)

- **Quick question** → Chat
- **Complex thinking** → Opus 4.7
- **Speed priority** → Sonnet 4.6
- **Document work** → Cowork
- **Building code/apps** → Claude Code
- **Slides/presentations** → Gamma
- **Visual mockups** → Claude Design
- **Need to act on web** → Chrome extension
- **System automation** → Computer Use
