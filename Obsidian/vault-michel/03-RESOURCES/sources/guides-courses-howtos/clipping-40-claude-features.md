# 40 Claude Features Most Users Never Touched

**@eng_khairallah1**: varredura completa de features obscuras do ecossistema Claude — web, desktop, API, Claude Code, Cowork.

## Estrutura

**Part 1 — Claude Chat (1-12):**
- Custom Instructions (system prompt permanente)
- Memory cross-conversation
- Extended Thinking (problema complexo → análise profunda)
- Artifacts (apps funcionais no chat)
- Projects (contexto persistente)
- Vision/image upload
- Voice mode
- File uploads além de texto
- Share conversations
- Model switching dentro do chat
- Haiku para tasks simples
- Prompt caching automático

**Part 2 — Claude Code (13-26):** hooks, skills, subagents, plugins, MCP, slash commands, CLAUDE.md layers, preview mode

**Part 3 — API (27-33):** function calling, tool use, streaming, token counting, caching, batch API

**Part 4 — Cowork (34-40):** role-specific plugins, connectors, sub-agents de produtividade, knowledge plugins

## Diferença de Resultados
