# Claude Can Do All of This — Feature Guide

**@AnatoliKopadze**: guide completo de features Claude que a maioria não conhece. Cobre web app, desktop, API, Claude Code, Cowork, e ecossistema.

## Features por Categoria

**Claude Chat:**
- **Projects**: contexto persistente entre conversas. Primeira coisa a configurar.
- **Artifacts**: apps funcionais dentro do chat (calculadoras, dashboards, jogos)
- **Custom Instructions**: system prompt permanente injetado em toda conversa
- **Memory**: Claude lembra entre conversas (requer configuração ativa)
- **Extended Thinking**: modo de raciocínio profundo para problemas complexos

**Claude Code:**
- Agente de coding autônomo com filesystem access
- Hooks para automações determinísticas
- Skills para comportamentos reutilizáveis
- Plugins (MCP servers + slash commands + skills bundled)

**Cowork:**
- Claude como colega de trabalho, não chatbot
- Plugins de role específico (sales, marketing, legal, data)
- Sub-agents para tarefas paralelas

## Insight Central

"Meanwhile a small group of users is running automated workflows, building full applications, delegating entire workdays to Claude... The difference is not intelligence. It is setup."
