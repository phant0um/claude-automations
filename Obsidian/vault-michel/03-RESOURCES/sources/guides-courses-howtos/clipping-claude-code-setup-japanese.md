# Claude Code 20 Setup Tips (Japanese)

**@MakeAI_CEO**: 20 configurações para reduzir alucinações e criar ambiente de AI coding real. Em japonês.

## Tese Principal

> "Diferença entre quem consegue resultados e quem não consegue com Claude Code não é qualidade do prompt — é profundidade do setup."

Novo engenheiro: não funciona sem onboarding. Claude Code: não funciona sem contexto. Solução: criar ambiente onde Claude erre menos.

## Os 20 Setups (estrutura inferida)

**Capítulo 1: Setup-first mindset**
- CLAUDE.md como instrumento para comportamento correto
- Rules, permissions, hooks, skills, subagents, plugins = dispositivos para "Claude se comportar corretamente"
- Alucinação ocorre por falta de informação, não por falha do modelo

**Capítulo 2: CLAUDE.md certo** (itens 1-5)
- Não dump de conhecimento geral (Claude já sabe Python/React)
- Regras de projeto específicas, naming conventions, arquivos proibidos

**Capítulo 3: Permissions e Hooks** (itens 6-10)
- Permitir Bash específico para o projeto
- Hooks deterministicamente garantem comportamento

**Capítulos 4-7: Skills, Subagents, Plugins, Retrospectiva** (itens 11-20)
- Cada ferramenta adicional aprofunda o contexto e consistência

## Valor para o Vault

Perspectiva japonesa do mesmo padrão (setup > prompts). Confirma convergência global de best practices.
