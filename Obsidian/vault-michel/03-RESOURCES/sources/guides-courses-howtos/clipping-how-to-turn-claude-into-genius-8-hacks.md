# How To Turn Claude Into A Genius (8 Hacks)

**Author:** Miles Deutscher (@milesdeutscher) | **Published:** 2026-03-16

## Summary

Eight fixes for improving Claude output quality after reported "nerfing." Core insight: Claude is a suite of tools (Chat, Cowork, Code, Projects) — match tool to task. Covers correct model selection (Haiku=fast/easy, Sonnet=daily, Opus=deep reasoning), extended thinking for complex tasks, structured prompting, and project-based memory. Emphasizes that most quality issues stem from using the wrong tool or insufficient context.

## Key Takeaways
- Match Claude tool to task: Chat (one-offs), Cowork (local files/automations), Code (building), Projects (memory layer)
- Model selection matters: Haiku for speed, Sonnet for daily use, Opus for reasoning
- Extended thinking mode unlocks deeper analysis on complex tasks
- Claude Projects provide persistent memory across sessions
- Most "quality degradation" complaints stem from insufficient context or wrong tool choice
