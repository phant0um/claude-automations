# Zero to AI Engineer — The Roadmap

**@seelffff**: cancelou $300/mês em cursos pagos. Tornou-se AI engineer com recursos gratuitos (Google, Anthropic, OpenAI, GitHub). 14 semanas, passo a passo.

## O Insight

Empresas que *constroem* AI (Google, Anthropic, OpenAI) passaram a dar treinamento de graça. Não vídeos de intro — cursos completos com certificados. GitHub tem repos de 95k+ stars que ensinam melhor que qualquer curso pago.

## Regras

1. Não pule etapas — Step 3 assume Step 2 feito
2. Anote (usa Obsidian, local, markdown). Após cada sessão: o que aprendi / o que surpreendeu / o que ainda não entendo
3. Construa em cada step — checkpoint funcional antes de avançar

## Estrutura de 14 Semanas (inferida da pasta Obsidian)

```
00-setup/           → contas + ferramentas
01-fundamentals/    → Python, math basics
02-ml-foundations/  → supervised/unsupervised, gradients
03-deep-learning/   → neural nets, Karpathy exercises
04-llms-prompting/  → APIs, prompt engineering
05-agents/          → tools, memory, harness
06-deployment/      → vercel, cloud, production
07-projects/        → portfolio de demonstração
```

## Recursos Gratuitos Chave

- Anthropic courses (Claude API, MCP, agent building)
- OpenAI cookbook
- Google ML crash course
- fast.ai
- Andrej Karpathy YouTube (makemore, nanoGPT)
