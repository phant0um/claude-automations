# Suryansh Tiwari — 9 Claude Prompts for Solo Founders

> [!key-insight] Core point
> Framework de 9 prompts para usar Claude como stack operacional completo de uma startup solo, substituindo equipe e múltiplas ferramentas.

## Conteúdo

9 roles Claude assume para solo founders:

1. **CEO Assistant** — founder dashboard diário: prioridades, gargalos, decisões simplificadas
2. **Product Strategist** — MVP vs features desnecessárias, path to launch, monetização, retention loops
3. **Content Team** — X thread + LinkedIn + newsletter + viral hooks a partir de um tópico
4. **Research Analyst** — market trends, competidores, oportunidades ocultas em formato de relatório acionável
5. **Growth Hacker** (implícito no thread) — crescimento low-cost high-leverage
6. **Design + UX Reviewer** — feedback brutalmente honesto sobre landing page e UX (clareza, confiança, conversão)
7. **Chief of Staff** — weekly operating system: prioridades, blocos de foco, automações AI
8. **Startup Simulator** — board meeting simulado (CEO + investidor + product lead + growth lead + ops + cliente cético) → decisão final
9. **Leverage Multiplier** — auditoria do que automatizar, parar, substituir por AI, maior ROI
