## Tese central

Cada sessão Claude começa com memória zero — CLAUDE.md é a solução permanente. 21 instruções cobrem voz, comportamento, contexto de empresa, e preferências de output. Acessível para não-desenvolvedores.

## Setup (2 minutos)

1. Criar arquivo `CLAUDE.md` na pasta do projeto (maiúsculas exatas)
2. Colar instruções em texto simples
3. Salvar — Claude lê automaticamente em cada sessão daquela pasta

## Categorias das 21 instruções

**Parte 1 — Como Claude fala:**
- Eliminar filler phrases ("Great question!", "Of course!") → resposta começa direto
- Mostrar opções antes de agir (não escolher abordagem e executar unilateralmente)

**Parte 2 — Voz e estilo:**
- Definir tom (formal/casual, técnico/acessível)
- Exemplos de output preferido

**Parte 3 — Contexto de empresa/projeto:**
- Nome, produto, público-alvo, concorrentes
- Contexto que Claude precisaria saber para ser útil

**Parte 4 — Formato de output:**
- Estrutura preferida (bullets vs prosa, headers, tamanho)
- Língua preferida

**Parte 5 — Regras operacionais:**
- O que Claude NUNCA deve fazer
- Quando perguntar vs quando agir
