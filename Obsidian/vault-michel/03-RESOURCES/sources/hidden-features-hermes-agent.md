## Tese central
Coleção viva de features hidden do Hermes Agent compilada da comunidade. Inclui /handoff (transfer live conversation entre platforms, same session id), hermes -c (continue last session), context compression behavior (clamp count, kicks in ~50% full).

## Key insights
- /handoff: mesma session continua em telegram/discord/slack — não fork ou restart
- hermes -c: resume most recent CLI session with full history
- Context compression: auto-summarize ~50% full, clamp icon mostra count
