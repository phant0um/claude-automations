## Tese central
Agents fail not because can't reason — objectives, outcomes, constraints são underspecified. Intent Engineering Framework: structure objectives, outcomes, health metrics, constraints, decision authority, stop rules para reliable autonomy. Solução não é more detailed instructions.

## Key insights
- Intent = objectives + outcomes + health metrics + constraints + decision authority + stop rules
- "The solution isn't more detailed instructions"
- Agent optimiza o que você especifica, não o que você meant
