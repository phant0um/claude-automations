# LLMs Improving LLMs

> Fonte específica não identificada — tema geral self-improvement (Self-Improve, AlphaLLM e similares).

## Resumo

Linha de pesquisa: LLM gera dados de treino/feedback pra si mesmo (self-distillation, self-critique, RL com reward model próprio) — exemplos: STaR, Self-Improve, AlphaLLM (busca + auto-avaliação). Eixo central: agente melhora performance sem dado humano novo, via auto-geração + filtro de qualidade.
