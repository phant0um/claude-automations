## Tese central
Agents relêem seu mundo repetidamente — ineficiência de context management. Análise do padrão e suas causas.

## Key insights
- Rereading = context não persistido entre steps
- Causa: falta de memory/state entre tool calls
- Oportunidade: melhor memory reduces redundant reads
