# Obsidian Sync — 11 Métodos Comparados (Chinese)

**@KyrieCheungYep**: comparação completa dos 11 principais métodos de sincronização multi-device para Obsidian. Em chinês.

## Por Que É Difícil

- Vault = pasta com centenas/milhares de arquivos pequenos
- iOS é o maior gargalo — soluções que funcionam no desktop travam no iPhone
- Mobile + multi-device + real-time: difícil ter os três ao mesmo tempo
- Conflitos silenciosos, arquivos sumindo sem aviso

## 11 Métodos em 7 Categorias

**A · Oficial:**
1. **Obsidian Sync** — $8-10/mês, mais simples, histórico de versão

**B · Cloud Drive:**
2. iCloud Drive — Apple ecosystem
3. OneDrive — Windows + macOS
4. Google Drive — Desktop + Android
5. Dropbox — cross-platform, ruim no Brasil

**C · WebDAV:**
6. 坚果云 (Jianguoyun) + Remotely Save — melhor opção gratuita para usuários chineses

**D · Git:**
7. GitHub/GitLab + obsidian-git plugin — gratuito, versioning, sem iOS nativo

**E · Self-hosted:**
8. Syncthing — P2P local, sem cloud
9. rsync/SSH — técnico, muito controle

**F · App bridges:**
10. Working Copy (iOS) + iSH

**G · VPS/reverse proxy:**
11. Custom server + WebDAV

## Decisão Framework

5 perguntas antes de escolher:
1. Usa iOS?
2. Precisa de tempo real?
3. Compartilha vault?
4. Orçamento?
5. Habilidade técnica?
