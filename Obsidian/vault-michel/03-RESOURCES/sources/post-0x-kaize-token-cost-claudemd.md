# Post @0x_kaize — Queimando $200/mês no Claude? O problema é você

> [!tip] Insight central
> 5 linhas no CLAUDE.md cortaram 60% dos gastos mensais com tokens — o problema não é o modelo, é o histórico de conversa reenviado integralmente a cada chamada.

## Conteúdo

Cada chamada de API reenvia o **histórico completo** da conversa:
- 1 mensagem = ~500 tokens
- 20 mensagens = ~200.000 tokens (pagos toda vez)

Solução: hábitos de economia de tokens no CLAUDE.md. Mesmo 2–3 hábitos já geram redução expressiva de custo.
