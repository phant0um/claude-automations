# Post @DaveJ — Documentar fluxos de app como HTML + JSON

> [!tip] Insight central
> Pedir ao Claude para gerar uma página HTML interativa de documentação de fluxos (alimentada por JSON) é útil tanto para humanos quanto como contexto para novos recursos/bug fixes.

## Técnica

Prompt sugerido: gerar página HTML única que mostra fluxos entre pacotes/componentes, com botões para diferentes ações (ex: "Convidar usuário", "Build para desktop"), destacando o caminho completo do fluxo e anotando o que é passado entre pacotes. Alimentado por JSON que documenta todos os fluxos.

**Duplo uso:** leitura humana + ingestão de contexto por LLM em sessões futuras.
