# Post @Suryanshti777 — Pare de pedir Markdown. Peça HTML.

> [!tip] Insight central
> Trocar Markdown por HTML como output de IA transforma o modelo de chatbot que cospe texto em parceiro que constrói interfaces — a interface de IA do futuro não é chat, é software gerado.

## Argumento central

Markdown era útil quando modelos eram fracos. Modelos atuais podem criar ferramentas, painéis, simulações, fluxos visuais. Pedir MD desperdiça essa capacidade.

**Em vez de receber:** 2.000 linhas de markdown, specs gigantes, docs bagunçados.

**Você recebe:** explicadores visuais, protótipos interativos, diffs anotados, quadros de planejamento arrastáveis, diagramas SVG, prévias ao vivo.

## Exemplos práticos

- Interface de repriorização de tickets: arrastar-e-soltar, colunas Agora/Próximo/Depois, botão de exportação para Claude Code
- Explicador de rate limiter: diagramas de token bucket, trechos anotados, decomposições de fluxo

## Tese maior

> "A interface de IA do futuro provavelmente não é chat. É software gerado."

O problema do output IA "de baixa qualidade" não é inteligência — é apresentação. Humanos entendem visualmente.
