# 25 Skills de productividad para Claude, ChatGPT y Gemini

> [!key-insight] Insight principal
> 25 skills em formato .md, organizadas em 5 categorias e agnósticas de modelo — funcionam em Claude, ChatGPT e Gemini colando o conteúdo como contexto/instrução.

## Content summary

### 5 Categorias e skills por categoria

| Categoria | Skills (#) |
|-----------|-----------|
| Estudantes | 1-7: apuntes, exámenes, roadmaps, explicador, TFG, flashcards, planificador |
| Productividade Laboral | 8-13: emails, reuniões/actas, CV+LinkedIn, apresentações |
| Investigação e Análise | 12-15: síntese de investigação, validação de fontes, estruturação de conhecimento, inteligência competitiva |
| Vídeo e Conteúdo Visual | 16-18: guions de vídeo, hooks, diagramas de fluxo |
| Código e Automatização | 19-25: documentador de código, tests unitários, debugger, regex, commits convencionais, code review, automação de workflows |

### Skills mais detalhadas (full SKILL.md)

**Para estudantes:**
- `structured-notes-generator` — PDF/transcrição → apuntes com conceitos, definições, exemplos, mapa de relações
- `exam-prep-generator` — apuntes → exame simulado (test, desenvolvimento, casos práticos) com respostas
- `learning-roadmap-generator` — "quero aprender X" → plano semanal com recursos, exercícios, checkpoints (máx 12 semanas)
- `flashcard-creator` — extrai pares pregunta-resposta para Anki/Quizlet

**Para código:**
- `conventional-commit-generator` — diff/descrição → mensagem Conventional Commits (type(scope): desc)
- `unit-test-generator` — função → suite completa (happy path + edge cases + erros) em Jest/Pytest/JUnit
- `debug-assistant` — erro + código → diagnóstico causa raiz + solução mínima + prevenção

### Como usar

- Em Claude: adicionar como Projeto ou colar como contexto na conversa
- Em ChatGPT: copiar bloco de código → instrução personalizada
- Em Gemini: colar como prompt de sistema/contexto
