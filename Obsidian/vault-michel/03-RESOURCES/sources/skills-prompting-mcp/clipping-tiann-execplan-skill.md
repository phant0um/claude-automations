# tiann/execplan-skill — Long-Running Task Skill

Skill que permite agentes de coding tacklear tasks complexas e de longa duração autonomamente. Inspirado em Codex Execution Plans (OpenAI Cookbook).

## O Que Oferece

- **Extended autonomous execution**: agente trabalha independentemente por horas sem perder contexto
- **Verifiable approach**: revise e aprove o plano antes da implementação começar
- **Living documentation**: progresso, decisões e descobertas trackados em real-time
- **Observable outcomes**: cada milestone produz resultado testável
- **Safe recovery**: rollback procedures built-in se algo der errado

## Compatibilidade

Claude Code, Cursor, OpenCode, Amp, Goose, VS Code — qualquer ferramenta Agent Skills-compatible.
