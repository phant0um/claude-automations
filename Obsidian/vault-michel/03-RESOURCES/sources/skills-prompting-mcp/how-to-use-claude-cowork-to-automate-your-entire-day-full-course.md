## Tese central

Claude Cowork é a versão do Claude Code para não-desenvolvedores: acessa arquivos locais, conecta Gmail/Calendar/Drive/Slack/Notion e executa tarefas agendadas — tudo via interface visual sem código. Anthropic o descreveu como "Claude Code for everyone else."

## Argumentos principais

- Cowork transforma Claude de chatbot em funcionário digital que opera no computador real do usuário, com acesso a arquivos reais e serviços conectados.
- O valor real não é uma tarefa única mas o sistema de automações encadeadas rodando enquanto o usuário trabalha em outras coisas.
- Dispatch mode permite controlar o agente no laptop a partir do celular — a tarefa roda localmente enquanto o usuário está em reunião.
- Zero código e zero setup técnico além de baixar o app.

## Key insights

- Rotina matinal automatizada: briefing diário gerado às 7:30am lendo Gmail, Calendar, Slack e /Projects — o que levaria 20-30 minutos manual é feito em 3 minutos.
- Cinco workflows de alto valor: Invoice Processor, Receipt Organizer, Report Compiler, Meeting Prep Automator, Content Batch Processor.
- Multi-step pipeline de 6 etapas para relatório de cliente: ler notes → buscar Slack → buscar Drive → compilar → salvar em Drive → draft email + post Slack — 3-5 minutos contra 45-60 manual.
- Cada conector leva 2-3 cliques para autorizar (Gmail, Calendar, Drive, Slack, Notion).
- Limite honesto: algumas tarefas precisam de iterações, arquivos muito grandes atingem limites de processamento.

## Exemplos e evidências

- Processamento de PDFs de fatura: 3 horas manual → 5 minutos Claude.
- Morning briefing: 20-30 minutos manual → 3 minutos Claude.
- Client report pipeline: 45-60 minutos manual → 3-5 minutos Claude.
- Dia 1 vs Dia 14: Dia 1 — 30 minutos de verificação manual antes de trabalhar; Dia 14 — lê briefing compilado overnight, começa trabalho real imediatamente.

## Implicações para o vault

- Claude Cowork é a porta de entrada para automação de agentes para usuários sem background técnico — entender seu modelo de produto explica decisões de design do Claude Desktop.
- O padrão de "morning briefing automatizado" é o mesmo usado no vault via CLAUDE.md + Filesystem MCP; Cowork democratiza isso com UI.
- Dispatch mode (controle remoto do agente) é um padrão que aparece também em outras fontes desta ingestão (remote control do Claude Code).
