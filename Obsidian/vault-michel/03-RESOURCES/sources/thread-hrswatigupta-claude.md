# Thread @hrswatigupta — 8 Claude Prompts to Replace Design Tools

Autor: [[03-RESOURCES/entities/hrswatigupta]] (Swati Gupta)

## Tese central

Claude + AI tools substituem Canva Pro e Figma para criação de designs. O thread apresenta 8 prompts estruturados com persona + tarefa + output esperado.

## 8 Prompts de Design com Claude

1. **Design Idea Generator** — Persona: senior UI/UX designer. Retorna 5 conceitos, layout, cor, tipografia.
2. **Social Media Post Creator** — Persona: graphic designer. Retorna headline, layout, visual style, CTA.
3. **Carousel Design System** — Persona: content designer. Retorna 7 slides com hook, direção visual, consistência de layout.
4. **Thumbnail / Cover Generator** — Persona: YouTube + social designer. Retorna hook text, visual idea, emotion trigger.
5. **Brand Style Guide** — Persona: brand strategist. Retorna paleta, fontes, tom de marca.
6. **Design Prompt for AI Tools** — Persona: prompt engineer. Converte ideia em prompt para Midjourney/DALL-E/Leonardo com style keywords.
7. **Landing Page Wireframe** — Persona: product designer. Retorna seções, content blocks, layout flow, CTA placement.
8. **Design Feedback + Improvement** — Persona: senior designer. Retorna erros, melhorias, versão revisada.

## Padrão de prompt observado

Todos os prompts seguem a estrutura: `Act as [persona]. [Tarefa com variável]. Return: [lista de outputs esperados]`.
Isso é o padrão Role + Task + Output do [[03-RESOURCES/concepts/prompt-engineering-patterns]].
