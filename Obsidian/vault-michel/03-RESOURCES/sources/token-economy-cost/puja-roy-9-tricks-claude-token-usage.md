# Imogen AI / puja_roy30dr — 9 Smart Tricks for Claude Token Usage

## Conteúdo

Essencialmente idêntico ao thread do @arceyul, mas em inglês e com 9 (vs 10) técnicas:

1. Editar prompt original em vez de acumular follow-ups
2. Nova conversa a cada 20 mensagens (resumo → novo chat)
3. Agrupar várias perguntas num único turno
4. Subir arquivos recorrentes a Projects (cache)
5. Configurar memória + custom instructions uma vez
6. Desativar funções não usadas (web search, research mode, connectors)
7. Usar Haiku para tarefas simples
8. Escolher modelo pela tarefa (Haiku/Sonnet/Opus)
9. Distribuir em 2-3 sessões diárias (janela de 5h)
