The most popular skills and skill registries in the Hermes ecosystem, ranked by GitHub stars. Skills are reusable capabilities that follow the agentskills.io open standard — from cybersecurity to diagram gene

## Argumentos principais
### Agents365-ai / drawio-skill
Enables the generation of editable.drawio diagrams from natural language prompts. It specializes in visualizing codebase structures for Python, JS, Go, and Rust using a library of 10,000+ cloud icons.

### conorbronsdon / avoid-ai-writing
An auditing tool that eliminates detectable AI writing patterns through a two-pass rewrite process. It utilizes a tiered replacement table and 49 pattern categories to remove structural and rhythmic AI fingerprints.

### wondelai / skills
A modular, cross-platform skills library compatible with Claude Code and agentskills.io agents. Skills are based on industry guidelines and can be installed via a plugin marketplace or npx.

### Romanescu11 / hermes-skill-factory
A meta-skill plugin that monitors active workflows to automatically detect and propose reusable skills. It generates Python plugin scaffolds and SKILL.md documentation, manageable via slash commands.

### DougTrajano / pydantic-ai-skills
Provides type-safe agentskills.io support for Pydantic AI using progressive disclosure to reduce token usage. It supports both programmatic and filesystem-based Python skill definitions.

### ZeroPointRepo / youtube-skills
A set of skills for extracting YouTube transcripts, captions, and timestamps without using yt-dlp. It supports video search and playlist extraction for Hermes Agent, OpenClaw, and Claude Code.

### Cranot / super-hermes
Teaches Hermes Agent to write its own analytical prompts using dynamic 'prism-scan' generation. It includes seven validated prisms for simulation and optimization, with a reflection tool to identify analysis blind spots.

### AkoliteZA / hermes-agent-idea-workflow
A workflow skill that converts rough ideas into structured design documents and implementation handoffs. It offers Lite and Full modes and includes a spec review gate with PASS/FAIL evaluation models.

### ReinaMacCredy / maestro
A harness for long-running agents featuring a durable card model for tracking tasks and decisions. It uses local-first storage in.maestro/ for diff reviews and requires proof/QA gates before closing cards.

### tlehman / litprog-skill
A literate programming skill that weaves prose, diagrams, and source code into.lit.md files. It includes a tangler to extract runnable source files and automatically syncs edits back to the literate document.

### esaradev / icarus-plugin
A self-memory system that captures high-value decisions and outcomes via shared markdown files. It provides a pipeline for fine-tuning and switching to replacement models based on captured work.

### smartcontractkit / chainlink-agent-skills
Implements agentskills.io specifications for interacting with Oracle networks and smart contracts. It supports CCIP, VRF, Data Feeds, and Data Streams, installable via the Vercel skills CLI.

### armelhbobdad / bmad-module-skill-forge
Converts repositories, documentation, and developer discourse into agentskills.io-compliant skills. It uses AST analysis to extract verified function signatures and links instructions to specific source code lines.

### black-forest-labs / skills
Official FLUX image generation skills providing standardized prompting guidelines for FLUX.1 and FLUX.2. It includes API integration patterns for webhooks and polling, supporting JSON structured prompting and hex color specs.

### chigwell / skilldock.io
A registry for reusable AI skills based on the AgentSkills specification. It features an OpenAPI-driven Python client for lifecycle management and supports monetization via TON and Stripe.
