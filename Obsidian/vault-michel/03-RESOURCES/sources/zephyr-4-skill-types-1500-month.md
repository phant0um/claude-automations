# Zephyr — Os 4 tipos de Claude Skills que vendem por $1.500+/mês

> [!key-insight] Core point
> Skills comerciais seguem 3 critérios (dor recorrente, julgamento codificado, invisibilidade) e se encaixam em 4 tipos estruturais — especificidade é o moat, genericidade é a morte.

## Conteúdo

### O que torna uma skill comercial
1. **Dor recorrente** — não "nice to have"; um fardo semanal que o comprador enfrenta ao acordar
2. **Julgamento codificado** — amostras de voz, rubrics de decisão, handling de edge cases; a skill sabe o que é "bom" para aquele comprador
3. **Invisibilidade** — comprador vê o output chegando na pasta, não o prompt rodando

### 4 Tipos de Skills comerciais

| Tipo | Função | Comprador | Preço |
|---|---|---|---|
| **Voice Replication** | Reescreve qualquer draft na voz da marca (20-30 amostras) | Agências, founders, course creators, marketing leads | $1.500-$2K setup + $200/mês |
| **Research Pipeline** | Lista de empresas/URLs → pesquisa estruturada em cadência definida | Sales teams, agências, founders em outbound | $1.200-$1.500 setup + $300/mês |
| **Customer Intelligence** | Ingere feedback não-estruturado (reviews, tickets, call transcripts) → padrões, linguagem, objeções | Marketing teams, founders, agências | $1.800-$2.500 por projeto |
| **Workflow Automation** | Substitui workflow manual recorrente que toca múltiplas ferramentas | Solo founders, executivos | $1.500 setup + $300/mês |

### O que a maioria faz errado
- Constroem skills genéricas ("funciona para todos") → vendem a $19 no Gumroad → comprador percebe que output = Claude cru
- **Especificidade é o moat** — skill que vende $1.500+/mês está errada para todo mundo exceto seu buyer profile específico
