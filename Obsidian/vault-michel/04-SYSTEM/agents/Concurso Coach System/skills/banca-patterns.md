# Padrões de Cobrança — CESPE, FGV, FCC

## CESPE / CEBRASPE

**Formato:** Certo/Errado (C/E). Erro derruba acerto (–1 anula +1) em algumas provas.

**Lógica do examinador:**
- Frase parece correta mas tem 1 palavra distorcida
- Modificador absoluto onde a matéria admite exceção
- Doutrina minoritária apresentada como majoritária
- Recorte de jurisprudência com contexto removido

**Padrões frequentes:**

| Padrão | Sinal | Probabilidade ERRADO |
|--------|-------|----------------------|
| "somente X" em matéria com exceção | absolutismo | ~80% |
| "sempre" / "necessariamente" / "em qualquer caso" | absolutismo | ~75% |
| Troca de "pode" por "deve" (ou vice-versa) | obrigatoriedade vs faculdade | ~70% |
| "É vedado" sem ressalva legal existente | omissão de exceção | ~85% |
| Inversão de competência (União↔Estado, Lei Ord↔LC) | competência | ~90% |
| Prazo trocado (5 anos vs 10 anos) | prazo | ~95% |
| "Independentemente de" quando há requisito | omissão | ~80% |

**Tática de prova:**
1. Identifique o sujeito da frase
2. Identifique o verbo principal e seus modificadores
3. Confronte com texto literal de lei
4. Se houver modificador absoluto + exceção legal conhecida → ERRADO

## FGV

**Formato:** Múltipla escolha (5 alternativas A–E). Sem anulação por erro.

**Lógica do examinador:**
- Caso prático denso — exige interpretação sistemática
- Duas alternativas "corretas", uma é "a mais adequada"
- Cobra interpretação de normas combinadas (CF + lei + jurisprudência)
- Distratores construídos com base em erros típicos de candidato

**Padrões frequentes:**

| Padrão | Sinal | Como atacar |
|--------|-------|-------------|
| Enunciado de 8+ linhas com caso | densidade | Sublinhar fatos relevantes, ignorar enfeite |
| 2 alternativas "corretas" | precisão | A que usa termo técnico literal vence |
| "Conforme entendimento do STJ" | jurisprudência | Memorizar súmulas + repetitivos |
| Alternativa com nuance sutil | distrator | Eliminar absolutos primeiro |
| Enunciado com pergunta no fim | direcionamento | Reler pergunta após cada alternativa |

**Tática de prova:**
1. Leia a pergunta ANTES do caso
2. Leia o caso marcando dados objetivos (datas, valores, normas)
3. Elimine alternativas absurdas (geralmente 2)
4. Entre as 3 restantes, busque a mais técnica/literal

## FCC

**Formato:** Múltipla escolha (5 alternativas A–E). Sem anulação por erro.

**Lógica do examinador:**
- Letra de lei pura — quase decoreba
- Alternativa errada troca 1 palavra-chave do texto legal
- Lacuna para completar com termo exato
- Decoreba de prazos, percentuais, competências

**Padrões frequentes:**

| Padrão | Sinal | Como atacar |
|--------|-------|-------------|
| Alternativa = parágrafo literal de lei | letra de lei | Memorizar artigos centrais |
| "Conforme dispõe a Lei X" | citação | Comparar com texto literal |
| Lacuna ___ no enunciado | completar | Termo exato da lei vence |
| Prazos 5/10/30/60/90/180/365 dias | decoreba | Tabela de prazos por matéria |
| Percentuais (20%, 25%, 50%, 75%) | decoreba | Tabela de %s por matéria |

**Tática de prova:**
1. Identifique a lei/norma cobrada
2. Se souber o texto literal → resposta direta
3. Se não souber → eliminar alternativas com termo "estranho" à norma
4. Última saída: chute orientado pela alternativa mais técnica

## Comparativo síntese

|  | CESPE | FGV | FCC |
|--|-------|-----|-----|
| Estilo | C/E armadilha | Caso denso | Letra de lei |
| Foco do estudo | Doutrina + jurisprudência | Sistematização + casos | Memorização de artigos |
| Inimigo principal | Modificador absoluto | Distrator técnico | Palavra-chave trocada |
| Material decisivo | Manuais doutrinários + súmulas | Códigos comentados | Lei seca anotada |
| Tipo de revisão | Mapas mentais por exceções | Esquemas comparativos | Flashcards de artigos |

## Regras transversais

1. **Identifique a banca antes de responder** — muda completamente a abordagem
2. **CESPE: marque modificadores** — toda análise CESPE precisa apontar absolutos
3. **FGV: leia pergunta primeiro** — caso é distração
4. **FCC: confronte com letra de lei** — não interprete, decore
5. **Em qualquer banca: prazo trocado = erro quase certo**
