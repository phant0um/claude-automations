# Skill: okr-format

Framework OKR (Objectives and Key Results) — ciclo trimestral.

## Estrutura

### Objetivo (O)
- Qualitativo, inspirador, direcionador
- Responde: "O que quero alcançar?"
- 1-3 objetivos por trimestre (máx.)
- Formato: frase afirmativa no infinitivo ou presente

### Key Results (KR)
- Quantitativos, mensuráveis, com prazo implícito (fim do trimestre)
- Responde: "Como saberei que cheguei lá?"
- 2-4 KRs por objetivo
- Formato: verbo + métrica + valor-alvo (ex: "Publicar 12 threads no X com média de 50 engajamentos")

## Exemplo

**Objetivo:** Construir presença técnica reconhecível no X
- KR1: Publicar 12 threads sobre IA/tech em 90 dias
- KR2: Chegar a 500 seguidores orgânicos até 30/06
- KR3: Ter pelo menos 1 thread com 100+ engajamentos

## Grading

| Grade | Significado |
|-------|-------------|
| 0.0–0.3 | Falhou — revisar premissas |
| 0.4–0.6 | Progresso parcial — analisar bloqueios |
| 0.7–0.9 | Zona ideal — ambição calibrada |
| 1.0 | Meta fácil demais — subiu o nível para o próximo ciclo |

> Grade 0.7 é o target — OKR que chega em 1.0 toda vez não era ambicioso o suficiente.

## Ciclo trimestral

| Semana | Ação |
|--------|------|
| Semana 1 do trimestre | Definir OKRs com Norte |
| Fim de cada mês | Check-in: progresso + bloqueios |
| Última semana do trimestre | Grade + retrospectiva |
| Primeira semana do próximo | Definir novo ciclo |

## Separação OKR ↔ GTD

- **Norte (OKR):** define O QUE alcançar no trimestre
- **Eixo (GTD):** define COMO — projetos e next actions que servem os KRs
- Cada KR deve ter pelo menos 1 projeto ativo no GTD
